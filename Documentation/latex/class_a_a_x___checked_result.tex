\doxysubsection{Detailed Description}
Error checker convenience class for A\+A\+X\+\_\+\+Result

Implicitly convertable to an A\+A\+X\+\_\+\+Result.

Provides an overloaded {\ttfamily \mbox{\hyperlink{class_a_a_x___checked_result_af95bea6f011d9a779170fc6128dc7295}{operator=()}}} which will throw an \mbox{\hyperlink{class_a_a_x_1_1_exception_1_1_result_error}{A\+A\+X\+::\+Exception\+::\+Result\+Error}} if assigned a non-\/success result.

\begin{DoxyWarning}{Warning}
Never use this class outside of an exception catch scope
\end{DoxyWarning}
If the host supports A\+A\+X\+\_\+\+T\+R\+A\+CE tracing, a log is emitted when the exception is thrown. A stacktrace is added if the host\textquotesingle{}s trace priority filter level is set to k\+A\+A\+X\+\_\+\+Trace\+\_\+\+Priority\+\_\+\+Lowest

When an error is encountered, \mbox{\hyperlink{class_a_a_x___checked_result}{A\+A\+X\+\_\+\+Checked\+Result}} throws an A\+A\+X\+\_\+\+Checked\+Result\+::\+Exception exception and clears its internal result value.
