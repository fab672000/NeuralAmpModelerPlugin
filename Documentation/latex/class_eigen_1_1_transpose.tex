Expression of the transpose of a matrix. 


\begin{DoxyTemplParams}{Template Parameters}

\hline

It is the return type of \mbox{\hyperlink{class_eigen_1_1_dense_base_aa08e5f8065d44eb64cc58ead571660e2}{Matrix\+Base\+::transpose()}} and \mbox{\hyperlink{class_eigen_1_1_matrix_base_a703c22c924072c342fc95b0efb1468a8}{Matrix\+Base\+::adjoint()}} and most of the time this is the only way it is used.
