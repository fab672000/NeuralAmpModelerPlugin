Returns a new canvas that will generate S\+VG commands from its draw calls, and send them to the provided stream. Ownership of the stream is not transfered, and it must remain valid for the lifetime of the returned canvas.

The canvas may buffer some drawing calls, so the output is not guaranteed to be valid or complete until the canvas instance is deleted.

The \textquotesingle{}bounds\textquotesingle{} parameter defines an initial S\+VG viewport (view\+Box attribute on the root S\+VG element).
