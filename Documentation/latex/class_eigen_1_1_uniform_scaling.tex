\doxysubsection{Detailed Description}
Represents a generic uniform scaling transformation. 

\textbackslash{}geometry\+\_\+module


\begin{DoxyTemplParams}{Template Parameters}
{\em Scalar\+\_\+} & the scalar type, i.\+e., the type of the coefficients.\\
\hline
\end{DoxyTemplParams}
This class represent a uniform scaling transformation. It is the return type of Scaling(\+Scalar), and most of the time this is the only way it is used. In particular, this class is not aimed to be used to store a scaling transformation, but rather to make easier the constructions and updates of \mbox{\hyperlink{class_eigen_1_1_transform}{Transform}} objects.

To represent an axis aligned scaling, use the \mbox{\hyperlink{class_eigen_1_1_diagonal_matrix}{Diagonal\+Matrix}} class.
