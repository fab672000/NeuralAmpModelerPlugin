\begin{DoxyEnumerate}
\item The first layer (bottom most) just has a blur and translate

The fast bounds functions are used to enable the paint to be culled early in the drawing pipeline. If a subclass can support this feature it must return true for the \mbox{\hyperlink{class_sk_draw_looper_a63480c43ddb90db22b712acccb562688}{can\+Compute\+Fast\+Bounds()}} function. If that function returns false then compute\+Fast\+Bounds behavior is undefined otherwise it is expected to have the following behavior. Given the parent paint and the parent\textquotesingle{}s bounding rect the subclass must fill in and return the storage rect, where the storage rect is with the union of the src rect and the looper\textquotesingle{}s bounding rect. \mbox{\Hypertarget{class_sk_draw_looper_ad835f1d5553a02b35caa96ec23ba105d}\label{class_sk_draw_looper_ad835f1d5553a02b35caa96ec23ba105d}}
