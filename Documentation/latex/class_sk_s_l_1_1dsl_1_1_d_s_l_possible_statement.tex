\doxysubsection{Detailed Description}
Represents a \mbox{\hyperlink{class_sk_s_l_1_1_statement}{Statement}} which may have failed and/or have pending errors to report. Converting a Possible\+Statement into a \mbox{\hyperlink{class_sk_s_l_1_1_statement}{Statement}} requires \mbox{\hyperlink{class_sk_s_l_1_1_position_info}{Position\+Info}} so that any pending errors can be reported at the correct position.

Possible\+Statement is used instead of \mbox{\hyperlink{class_sk_s_l_1_1_statement}{Statement}} in situations where it is not possible to capture the \mbox{\hyperlink{class_sk_s_l_1_1_position_info}{Position\+Info}} at the time of \mbox{\hyperlink{class_sk_s_l_1_1_statement}{Statement}} construction.
