\doxysubsection{Detailed Description}
This class is used to support A\+Trace in android apps. It hooks into the \mbox{\hyperlink{class_sk_event_tracer}{Sk\+Event\+Tracer}} system. It currently supports the macros T\+R\+A\+C\+E\+\_\+\+E\+V\+E\+N\+T$\ast$, T\+R\+A\+C\+E\+\_\+\+E\+V\+E\+N\+T\+\_\+\+I\+N\+S\+T\+A\+N\+T$\ast$, and T\+R\+A\+C\+E\+\_\+\+E\+V\+E\+N\+T\+\_\+\+B\+E\+G\+I\+N/\+E\+N\+D$\ast$. For versions of these calls that take additoinal args and value pairs we currently just drop them and report only the name. Since A\+Trace is a simple push and pop system (all traces are fully nested), if using B\+E\+G\+IN and E\+ND you should also make sure your calls are properly nested (i.\+e. if startA is before startB, then endB is before endA).
