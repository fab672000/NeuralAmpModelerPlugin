{\footnotesize\ttfamily uint32\+\_\+t Gr\+Context\+\_\+\+Base\+::context\+ID (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [protected]}}

An identifier for this context. The id is used by all compatible contexts. For example, if Sk\+Images are created on one thread using an image creation context, then fed into a D\+DL Recorder on second thread (which has a recording context) and finally replayed on a third thread with a direct context, then all three contexts will report the same id. It is an error for an image to be used with contexts that report different ids.
