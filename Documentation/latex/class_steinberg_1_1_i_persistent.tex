\doxysubsection{Detailed Description}
Persistent Object Interface. \mbox{[}plug imp\mbox{]} ~\newline
This interface is used to store/restore attributes of an object. An I\+Plug\+Controller can implement this interface to handle presets. The gui-\/xml for a preset control looks like this\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{....}
\DoxyCodeLine{<view name=\textcolor{stringliteral}{"PresetView"} data=\textcolor{stringliteral}{"Preset"}/>}
\DoxyCodeLine{....}
\DoxyCodeLine{<\textcolor{keyword}{template} name=\textcolor{stringliteral}{"PresetView"}>}
\DoxyCodeLine{    <view name=\textcolor{stringliteral}{"preset control"} size=\textcolor{stringliteral}{"0, 0, 100, 20"}/>}
\DoxyCodeLine{    <\textcolor{keywordflow}{switch} name=\textcolor{stringliteral}{"store preset"} size=\textcolor{stringliteral}{"125,0,80,20"} style=\textcolor{stringliteral}{"push|immediate"} title=\textcolor{stringliteral}{"Store"}  />}
\DoxyCodeLine{    <\textcolor{keywordflow}{switch} name=\textcolor{stringliteral}{"remove preset"} size=\textcolor{stringliteral}{"220,0,80,20"} style=\textcolor{stringliteral}{"push|immediate"} title=\textcolor{stringliteral}{"Delete"}  />}
\DoxyCodeLine{</\textcolor{keyword}{template}>}
\end{DoxyCode}
 The tag data=\char`\"{}\+Preset\char`\"{} tells the host to create a preset controller that handles the 3 values named \char`\"{}preset control\char`\"{}, \char`\"{}store preset\char`\"{}, and \char`\"{}remove preset\char`\"{}.
