class Eigen\+::\+Matrix\+Square\+Root\+Return\+Value$<$ Derived $>$}

Proxy for the matrix square root of some matrix (expression). 


\begin{DoxyTemplParams}{Template Parameters}
{\em Derived} & Type of the argument to the matrix square root.\\
\hline
\end{DoxyTemplParams}
This class holds the argument to the matrix square root until it is assigned or evaluated for some other reason (so the argument should not be changed in the meantime). It is the return type of \mbox{\hyperlink{group__core__func__exponential_ga5c784ec51a1765be0a623104db2577b6}{Matrix\+Base\+::sqrt()}} and most of the time this is the only way it is used.
