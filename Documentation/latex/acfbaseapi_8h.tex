\doxysubsection{Detailed Description}
Defines the common public interfaces that must be implemented or used by all A\+CF plugins. 

\begin{DoxyRemark}{Remarks}

Opitional callback to the plug-\/in to allow plug-\/in module perform global cleanup. 

{\bfseries{A\+C\+F\+Shutdown}} \begin{DoxyRemark}{Remarks}
This opitional callback if it exists will be called by the A\+CF Host before a plug-\/in module is unloaded. Use this routine to cleanup any global state or cached component services used by the plug-\/in or any of plug-\/in components. It must be safe for the host to unload the plug-\/in module after the calling A\+C\+F\+Shutdown (i.\+e. \mbox{\hyperlink{acfbaseapi_8h_abfd77592e2f250a995a8e93237a5f54b}{A\+C\+F\+Can\+Unload\+Now()}} must return A\+C\+F\+\_\+\+OK). 
\end{DoxyRemark}
\begin{DoxyNote}{Note}
The A\+C\+F\+Shutdown is never called during the registration process; it is only called by the host after the plug-\/in module has been loaded to create components. 

This function may be exported from any A\+CF plug-\/in. The Host uses this callback to provide an optional hook for plug-\/ins to safely cleanup global state. 
\end{DoxyNote}

Opitional callback to the plug-\/in to allow plug-\/in module perform global initialization. 

{\bfseries{A\+C\+F\+Startup}} \begin{DoxyRemark}{Remarks}
This opitional callback if it exists will be called once by the A\+CF Host before the first call to A\+C\+F\+Get\+Class\+Factory. Use this routine to initialize any global state or services required by the plug-\/in and any plug-\/in components. 
\end{DoxyRemark}
\begin{DoxyNote}{Note}
The plug-\/in module may have been unloaded after registration or loaded without registration before A\+C\+F\+Startup is called. A\+C\+F\+Startup is only called after a plug-\/in any its components have been registered. 

This function may be exported from any A\+CF plug-\/in. The Host uses this callback to provide an optional hook for plug-\/ins to safely initialize global state. 
\end{DoxyNote}
