\doxysubsection{Detailed Description}
\subsubsection*{template$<$Sk\+Alpha\+Type k\+AT$>$\newline
struct Sk\+R\+G\+B\+A4f$<$ k\+A\+T $>$}

R\+G\+BA color value, holding four floating point components. Color components are always in a known order. k\+AT determines if the \mbox{\hyperlink{struct_sk_r_g_b_a4f}{Sk\+R\+G\+B\+A4f}}\textquotesingle{}s R, \mbox{\hyperlink{struct_g}{G}}, and B components are premultiplied by alpha or not.

Skia\textquotesingle{}s public A\+PI always uses unpremultiplied colors, which can be stored as \mbox{\hyperlink{struct_sk_r_g_b_a4f}{Sk\+R\+G\+B\+A4f$<$k\+Unpremul\+\_\+\+Sk\+Alpha\+Type$>$}}. For convenience, this type can also be referred to as \mbox{\hyperlink{struct_sk_color4f}{Sk\+Color4f}}.
