\doxysubsection{Detailed Description}
Light weight class for writing data to a memory block. The W\+Buffer is given the buffer to write into, with either a specified size or no size, in which case no range checking is performed. An empty W\+Buffer is legal, in which case no data is ever written, but the relative pos() is updated.
