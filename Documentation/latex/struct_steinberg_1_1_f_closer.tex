\subsubsection*{template$<$class T$>$\newline
struct Steinberg\+::\+F\+Closer$<$ T $>$}

Template definition for classes that help closing resources. A stack allocated object of this type automatically calls the close method of an at construction time passed object when it reaches the end of its scope. It goes without saying that the given type needs to have a close method. ~\newline
