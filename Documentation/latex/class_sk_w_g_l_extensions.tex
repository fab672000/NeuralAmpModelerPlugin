W\+GL doesn\textquotesingle{}t have precise rules for the ordering of formats returned by wgl\+Choose\+Pixel\+Format. This function helps choose among the set of formats returned by wgl\+Choose\+Pixel\+Format. The rules in decreasing priority are\+:
\begin{DoxyItemize}
\item Choose formats with the smallest sample count that is $>$= desired\+Sample\+Count (or the largest sample count if all formats have fewer samples than desired\+Sample\+Count.) If desired\+Sample\+Count is 1 then all msaa formats are excluded from consideration.
\item Choose formats with the fewest color samples when coverage sampling is available.
\item If the above rules leave multiple formats, choose the one that appears first in the formats array parameter. 
\end{DoxyItemize}
