\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_stream}{Sk\+Stream}} -- abstraction for a source of bytes. Subclasses can be backed by memory, or a file, or something else.

N\+O\+TE\+:

Classic \char`\"{}streams\char`\"{} A\+P\+Is are sort of async, in that on a request for N bytes, they may return fewer than N bytes on a given call, in which case the caller can \char`\"{}try again\char`\"{} to get more bytes, eventually (modulo an error) receiving their total N bytes.

Skia streams behave differently. They are effectively synchronous, and will always return all N bytes of the request if possible. If they return fewer (the \mbox{\hyperlink{class_sk_stream_afae3344ee277d55aec3a4fa5fead387a}{read()}} call returns the number of bytes read) then that means there is no more data (at E\+OF or hit an error). The caller should {\itshape not} call again in hopes of fulfilling more of the request.
