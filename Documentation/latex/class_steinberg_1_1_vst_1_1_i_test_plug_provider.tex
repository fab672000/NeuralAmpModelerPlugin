\doxysubsection{Detailed Description}
Test Helper.

This class provides access to the component and the controller of a plug-\/in when running a unit test (see \mbox{\hyperlink{class_steinberg_1_1_i_test}{I\+Test}}). You get this interface as the context argument in the \mbox{\hyperlink{class_steinberg_1_1_i_test_factory_a4078ea9dc6ccb06af158b8e0de738c29}{I\+Test\+Factory\+::create\+Tests}} method.
