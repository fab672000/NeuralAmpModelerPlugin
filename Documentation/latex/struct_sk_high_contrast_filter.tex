\doxysubsection{Detailed Description}
Color filter that provides transformations to improve contrast for users with low vision.

Applies the following transformations in this order. Each of these can be configured using \mbox{\hyperlink{struct_sk_high_contrast_config}{Sk\+High\+Contrast\+Config}}.


\begin{DoxyItemize}
\item Conversion to grayscale
\item Color inversion (either in R\+GB or H\+SL space)
\item Increasing the resulting contrast.
\end{DoxyItemize}

Calling Sk\+High\+Contrast\+Filter\+::\+Make will return nullptr if the config is not valid, e.\+g. if you try to call it with a contrast outside the range of -\/1.\+0 to 1.\+0.
