the smart pointer implementation 

\begin{DoxyVerb}A smart pointer is in charge of maintaining the objects reference count 
by the way of pointers operators overloading. It supports class 
inheritance and conversion whenever possible.
\end{DoxyVerb}
 ~\newline
 Instances of the \mbox{\hyperlink{classoscfaust_1_1_s_m_a_r_t_p}{S\+M\+A\+R\+TP}} class are supposed to use {\itshape smartable} types (or at least objects that implements the {\itshape add\+Reference} and {\itshape remove\+Reference} methods in a consistent way).
