\doxysubsection{Detailed Description}
Encapsulates the logic of deciding if we need to change/wrap the blitter for aaclipping. If so, get\+Rgn and get\+Blitter return modified values. If not, they return the raw blitter and (bw) clip region.

We need to keep the constructor/destructor cost as small as possible, so we can freely put this on the stack, and not pay too much for the case when we\textquotesingle{}re really BW anyways.
