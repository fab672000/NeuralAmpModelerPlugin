The faust code (.dsp) in this directory has been compiled using {\ttfamily faust2sam} into {\ttfamily sine\+\_\+synth-\/sam}


\begin{DoxyCode}{0}
\DoxyCodeLine{\% faust2sam -\/midi sine\_synth.dsp}
\end{DoxyCode}


The result of this compile is a zip file {\ttfamily sine\+\_\+synth-\/sam.\+zip} which contains the following three files\+:


\begin{DoxyItemize}
\item {\ttfamily fast\+\_\+pow2.\+h}
\item {\ttfamily sam\+Faust\+D\+S\+P.\+cpp}
\item {\ttfamily sam\+Faust\+D\+S\+P.\+h}
\end{DoxyItemize}

These 3 files should be placed into the directory {\ttfamily sam\+\_\+baremetal\+\_\+framework\+\_\+core1/src/faust}

In addition there is a header file that is common across all cores called {\ttfamily audio\+\_\+system\+\_\+config.\+h}. In this file the following pre-\/processor variables should be set in the following way. The example below indicates that a \mbox{\hyperlink{struct_faust}{Faust}} algorithm will only be running on Core1 and that Core2 will be simply passing audio to the codec.


\begin{DoxyCode}{0}
\DoxyCodeLine{\#define USE\_FAUST\_ALGORITHM\_CORE1           TRUE}
\DoxyCodeLine{\#define USE\_FAUST\_ALGORITHM\_CORE2           FALSE}
\end{DoxyCode}
