get\+Bounds places the current finite bound in its first parameter. In its second, it indicates which kind of bound is being returned. If \textquotesingle{}canv\+Finite\+Bound\textquotesingle{} is a normal bounding box then it encloses all writeable pixels. If \textquotesingle{}canv\+Finite\+Bound\textquotesingle{} is an inside out bounding box then it encloses all the un-\/writeable pixels and the true/normal bound is the infinite plane. is\+Intersection\+Of\+Rects is an optional parameter that is true if \textquotesingle{}canv\+Finite\+Bound\textquotesingle{} resulted from an intersection of rects. \mbox{\Hypertarget{class_sk_clip_stack_a4515940b0063ae9b0395baa7f45212ec}\label{class_sk_clip_stack_a4515940b0063ae9b0395baa7f45212ec}}

Get\+Conservative\+Bounds returns a conservative bound of the current clip. Since this could be the infinite plane (if inverse fills were involved) the max\+Width and max\+Height parameters can be used to limit the returned bound to the expected drawing area. Similarly, the offsetX and offsetY parameters allow the caller to offset the returned bound to account for translated drawing areas (i.\+e., those resulting from a save\+Layer). For finite bounds, the translation (+offsetX, +offsetY) is applied before the clamp to the maximum rectangle\+: \mbox{[}0,max\+Width) x \mbox{[}0,max\+Height). is\+Intersection\+Of\+Rects is an optional parameter that is true when \textquotesingle{}dev\+Bounds\textquotesingle{} is the result of an intersection of rects. In this case \textquotesingle{}dev\+Bounds\textquotesingle{} is the exact answer/clip. \mbox{\Hypertarget{class_sk_clip_stack_a1185947d0a0f56da2d774c3193017989}\label{class_sk_clip_stack_a1185947d0a0f56da2d774c3193017989}}
