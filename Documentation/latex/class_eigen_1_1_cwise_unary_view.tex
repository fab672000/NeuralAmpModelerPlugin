\subsubsection*{template$<$typename View\+Op, typename Matrix\+Type, typename Stride\+Type$>$\newline
class Eigen\+::\+Cwise\+Unary\+View$<$ View\+Op, Matrix\+Type, Stride\+Type $>$}

Generic lvalue expression of a coefficient-\/wise unary operator of a matrix or a vector. 


\begin{DoxyTemplParams}{Template Parameters}
{\em View\+Op} & template functor implementing the view \\
\hline
{\em Matrix\+Type} & the type of the matrix we are applying the unary operator\\
\hline
\end{DoxyTemplParams}
This class represents a lvalue expression of a generic unary view operator of a matrix or a vector. It is the return type of \mbox{\hyperlink{structreal}{real()}} and \mbox{\hyperlink{structimag}{imag()}}, and most of the time this is the only way it is used.
