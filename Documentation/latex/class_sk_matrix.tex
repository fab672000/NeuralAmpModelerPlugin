\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_matrix}{Sk\+Matrix}} holds a 3x3 matrix for transforming coordinates. This allows mapping \mbox{\hyperlink{struct_sk_point}{Sk\+Point}} and vectors with translation, scaling, skewing, rotation, and perspective.

\mbox{\hyperlink{class_sk_matrix}{Sk\+Matrix}} elements are in row major order. \mbox{\hyperlink{class_sk_matrix}{Sk\+Matrix}} constexpr default constructs to identity.

\mbox{\hyperlink{class_sk_matrix}{Sk\+Matrix}} includes a hidden variable that classifies the type of matrix to improve performance. \mbox{\hyperlink{class_sk_matrix}{Sk\+Matrix}} is not thread safe unless \mbox{\hyperlink{class_sk_matrix_af4d962619c7bde5e6f7666a36be043ce}{get\+Type()}} is called first.
