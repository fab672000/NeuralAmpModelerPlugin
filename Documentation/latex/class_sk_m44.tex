\doxysubsection{Detailed Description}
4x4 matrix used by \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} and other parts of Skia.

Skia assumes a right-\/handed coordinate system\+: +X goes to the right +Y goes down +Z goes into the screen (away from the viewer)

A matrix is categorized as \textquotesingle{}perspective\textquotesingle{} if the bottom row is not \mbox{[}0, 0, 0, 1\mbox{]}. For most uses, a bottom row of \mbox{[}0, 0, 0, X\mbox{]} behaves like a non-\/perspective matrix, though it will be categorized as perspective. Calling \mbox{\hyperlink{class_sk_m44_a5a9e16c2f9937add3634613de69180f9}{normalize\+Perspective()}} will change the matrix such that, if its bottom row was \mbox{[}0, 0, 0, X\mbox{]}, it will be changed to \mbox{[}0, 0, 0, 1\mbox{]} by scaling the rest of the matrix by 1/X.
