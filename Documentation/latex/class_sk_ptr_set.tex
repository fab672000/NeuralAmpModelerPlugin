\doxysubsection{Detailed Description}
Maintains a set of ptrs, assigning each a unique ID \mbox{[}1...N\mbox{]}. Duplicate ptrs return the same ID (since its a set). Subclasses can override in\+Ptr() and dec\+Ptr(). inc\+Ptr() is called each time a unique ptr is added ot the set. dec\+Ptr() is called on each ptr when the set is destroyed or reset.
