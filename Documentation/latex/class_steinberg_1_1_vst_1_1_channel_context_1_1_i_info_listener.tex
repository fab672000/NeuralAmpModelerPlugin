Allows the host to inform the plug-\/in about the context in which the plug-\/in is instantiated, mainly channel based info (color, name, index,...). Index can be defined inside a namespace (for example, index start from 1 to N for Type Input/\+Output Channel (Index namespace) and index start from 1 to M for Type Audio Channel).~\newline
As soon as the plug-\/in provides this \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_channel_context_1_1_i_info_listener}{I\+Info\+Listener}} interface, the host will call set\+Channel\+Context\+Infos for each change occurring to this channel (new name, new color, new indexation,...)\hypertarget{class_steinberg_1_1_vst_1_1_channel_context_1_1_i_info_listener_IChannelContextExample}{}\doxysubsection{Example}\label{class_steinberg_1_1_vst_1_1_channel_context_1_1_i_info_listener_IChannelContextExample}
