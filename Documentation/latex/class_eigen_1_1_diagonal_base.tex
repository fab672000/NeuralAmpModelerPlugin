class Eigen\+::\+Diagonal\+Base$<$ Derived $>$}

Base class for diagonal matrices and expressions. 

This is the base class that is inherited by diagonal matrix and related expression types, which internally use a vector for storing the diagonal entries. \mbox{\hyperlink{class_eigen_1_1_diagonal}{Diagonal}} types always represent square matrices.
