A context menu is composed of Item (entry). A Item is defined by a name, a tag, a flag and a associated target (called when this item will be selected/executed). With \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_context_menu}{I\+Context\+Menu}} the plug-\/in can retrieve a Item, add a Item, remove a Item and pop-\/up the menu.
