Ever wondered what 100 square waves sound like when played together? Well now you can find out!

Mega Drone is an app which creates 100 oscillators, combines their output in a mixer and plays the resulting sound.

This sample demonstrates how to obtain the lowest latency and optimal computational throughput by\+:

1) Leaving Oboe to choose the best default stream properties for the current device 2) Setting performance mode to Low\+Latency 3) Setting sharing mode to Exclusive 4) Setting the buffer size to 2 bursts 5) Using the {\ttfamily -\/Ofast} compiler optimization flag, even when building the {\ttfamily Debug} variant 6) Using \href{https://developer.android.com/reference/android/os/Process\#getExclusiveCores()}{\texttt{ {\ttfamily get\+Exclusive\+Cores}}} (A\+PI 24+) and thread affinity to bind the audio thread to the best available C\+PU core(s) 7) Using a {\ttfamily Stabilized\+Callback} which aims to spend a fixed percentage of the callback time to avoid C\+PU frequency scaling (\href{https://www.youtube.com/watch?v=C0BPXZIvG-Q&feature=youtu.be&t=1158}{\texttt{ video explanation}})

This code was presented at \href{http://www.aes.org/events/144/}{\texttt{ A\+ES Milan}} and \href{https://www.de.droidcon.com/}{\texttt{ Droidcon Berlin}} as part of a talk on Oboe.
