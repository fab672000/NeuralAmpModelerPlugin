\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Sk\+Span$<$ T $>$}

An \mbox{\hyperlink{class_sk_span}{Sk\+Span}} is a view of a contiguous collection of elements of type T. It can be directly constructed from a pointer and size. Sk\+Make\+Span can be used to construct one from an array, or a container (like std\+::vector).
