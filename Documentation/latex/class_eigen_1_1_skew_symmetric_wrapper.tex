\begin{DoxyTemplParams}{Template Parameters}
{\em Skew\+Symmetric\+Vector\+Type\+\_\+} & the type of the vector of coefficients\\
\hline
\end{DoxyTemplParams}
This class is an expression of a skew symmetric matrix, but not storing its own vector of coefficients, instead wrapping an existing vector expression. It is the return type of \mbox{\hyperlink{class_eigen_1_1_matrix_base_af75ae473f1a31ae8d2af4d07c1e84323}{Matrix\+Base\+::as\+Skew\+Symmetric()}} and most of the time this is the only way that it is used.
