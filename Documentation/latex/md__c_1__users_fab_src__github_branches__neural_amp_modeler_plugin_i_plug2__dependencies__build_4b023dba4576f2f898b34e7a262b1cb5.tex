The E\+S\+P32 is a cheap microcontroller developed and distributed by Espressif\+: \href{https://www.espressif.com/en/products/hardware/esp32/overview}{\texttt{ https\+://www.\+espressif.\+com/en/products/hardware/esp32/overview}}. It is used at the heart of an increasing number of boards, especially for real-\/time \mbox{\hyperlink{class_d_s_p}{D\+SP}} applications.

{\ttfamily faust2esp32 \mbox{[}-\/lib\mbox{]} \mbox{[}-\/gramophone\mbox{]} \mbox{[}-\/multi\mbox{]} \mbox{[}-\/midi\mbox{]} \mbox{[}-\/nvoices $<$num$>$\mbox{]} \mbox{[}-\/wm8978 or -\/ac101\mbox{]} $<$file.\+dsp$>$}

{\ttfamily faust2esp32} can be used to turn a \mbox{\hyperlink{struct_faust}{Faust}} program into an \mbox{\hyperlink{class_d_s_p}{D\+SP}} engine for the E\+S\+P32. While new options are being added to it, the current (and only) way to use {\ttfamily faust2esp32} is by running\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{faust2esp32 -\/lib file.dsp}
\end{DoxyCode}


which will generate a {\ttfamily .zip} package containing\+:


\begin{DoxyItemize}
\item a \mbox{\hyperlink{class_d_s_p}{D\+SP}} engine taking the form a C++ class (named after the name of the \mbox{\hyperlink{struct_faust}{Faust}} file, so {\ttfamily file} here)
\item a C++ object to configure the potential audio codec used on your E\+S\+P32-\/based board (currently, only the Wolfson \mbox{\hyperlink{class_w_m8978}{W\+M8978}} used on the \href{https://github.com/LilyGO/TTGO-TAudio}{\texttt{ T\+T\+GO T-\/\+Audio}} board is being supported but we\textquotesingle{}re in the process of adding new ones).
\end{DoxyItemize}

Here are the available options\+:


\begin{DoxyItemize}
\item {\ttfamily -\/lib to generate a package containing an object compatible with any E\+S\+P32 project}
\item {\ttfamily -\/gramophone to generate files for G\+R\+A\+ME \mbox{\hyperlink{class_gramophone}{Gramophone}}}
\item `-\/multi' to generate for G\+R\+A\+ME \mbox{\hyperlink{class_gramophone}{Gramophone}} in multi \mbox{\hyperlink{class_d_s_p}{D\+SP}} mode{\ttfamily  -\/}-\/midi to activate M\+I\+DI control{\ttfamily  -\/}-\/nvoices $<$num$>$\textquotesingle{} to produce a polyphonic self-\/contained \mbox{\hyperlink{class_d_s_p}{D\+SP}} with $<$num$>$ voices\`{}
\end{DoxyItemize}
