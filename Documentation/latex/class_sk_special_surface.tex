\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_special_surface}{Sk\+Special\+Surface}} is a restricted form of \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}} solely for internal use. It differs from \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}} in that\+:
\begin{DoxyItemize}
\item it can be backed by Gr\+Textures larger than \mbox{[} f\+Width, f\+Height \mbox{]}
\item it can\textquotesingle{}t be used for tiling
\item it becomes inactive once a snapshot of it is taken (i.\+e., no copy-\/on-\/write)
\item it has no generation ID 
\end{DoxyItemize}
