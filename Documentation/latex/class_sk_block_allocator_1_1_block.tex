{\footnotesize\ttfamily bool Sk\+Block\+Allocator\+::\+Block\+::resize (\begin{DoxyParamCaption}\item[{int}]{start,  }\item[{int}]{end,  }\item[{int}]{delta\+Bytes }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}

Resize a previously reserved byte range of offset \textquotesingle{}start\textquotesingle{} (inclusive) to \textquotesingle{}end\textquotesingle{} (exclusive). \textquotesingle{}delta\+Bytes\textquotesingle{} is the S\+I\+G\+N\+ED change to length of the reservation.

When negative this means the reservation is shrunk and the new length is (end -\/ start -\/ $\vert$delta\+Bytes$\vert$). If this new length would be 0, the byte range can no longer be used (as if it were released instead). Asserts that it would not shrink the reservation below 0.

If \textquotesingle{}delta\+Bytes\textquotesingle{} is positive, the allocator attempts to increase the length of the reservation. If \textquotesingle{}delta\+Bytes\textquotesingle{} is less than or equal to avail() and it was the last allocation in the block, it can be resized. If there is not enough available bytes to accommodate the increase in size, or another allocation is blocking the increase in size, then false will be returned and the reserved byte range is unmodified.
