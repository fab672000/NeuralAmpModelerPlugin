-- Note --

This mechanism assumes monolithic fonts (single data blob). Some web font providers may serve multiple font blobs, segmented for various unicode ranges, depending on user agent capabilities (woff, woff2). In that case, the embedder would need to advertise no user agent capabilities when fetching the U\+RL, in order to receive full font data.
