\subsubsection*{template$<$typename T, int32\+\_\+t Real\+Precision = 1000$>$\newline
class A\+A\+X\+\_\+\+C\+Range\+Taper\+Delegate$<$ T, Real\+Precision $>$}

A piecewise-\/linear taper conforming to \mbox{\hyperlink{class_a_a_x___i_taper_delegate}{A\+A\+X\+\_\+\+I\+Taper\+Delegate}}. 

This taper spaces a parameter\textquotesingle{}s real values between its minimum and maximum using a series of linear regions to create the full mapping between the parameter\textquotesingle{}s real and normalized values.

Here is an example of how this taper can be used\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{float} rangePoints[] = \{ 0.0, 1.0, 100.0, 1000.0, 2000.0 \};}
\DoxyCodeLine{\textcolor{keywordtype}{double} rangeSteps[] = \{ 0.1, 1.0, 10.0, 25.0 \}; \textcolor{comment}{// number of steps per range: 10, 99, 90, 40 }}
\DoxyCodeLine{\textcolor{keyword}{const} \textcolor{keywordtype}{long} cNumRanges = \textcolor{keyword}{sizeof}(rangeSteps)/\textcolor{keyword}{sizeof}(rangeSteps[0]);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{long} numSteps = 0;}
\DoxyCodeLine{\textcolor{keywordflow}{for} (\textcolor{keywordtype}{int} i = 0; i < cNumRanges; i++)}
\DoxyCodeLine{\{}
\DoxyCodeLine{    numSteps += (rangePoints[i+1] -\/ rangePoints[i]) / rangeSteps[i];}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{class_a_a_x___c_range_taper_delegate}{AAX\_CRangeTaperDelegate<float>}} nonLinearTaper(rangePoints, rangeSteps, cNumRanges);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{float} controlValue = 1.5;}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{double} normalized = nonLinearTaper.RealToNormalized(controlValue);}
\DoxyCodeLine{\textcolor{keywordtype}{float} \mbox{\hyperlink{structreal}{real}} = nonLinearTaper.NormalizedToReal(normalized);}
\end{DoxyCode}


\begin{DoxyParagraph}{Real\+Precision}
In addition to its type templatization, this taper includes a precision template parameter. Real\+Precision is a multiplier that works in conjunction with the \mbox{\hyperlink{group__core__func__common_gabf111d6915720188a3c891b476308e80}{round()}} function to limit the precision of the real values provided by this taper. For example, if Real\+Precision is 1000, it will round to the closest 0.\+001 when doing any sort of value conversion. If Real\+Precision is 1, it will round to the nearest integer. If Real\+Precision is 1000000, it will round to the nearest 0.\+000001. This is particularly useful for preventing things like 1.\+9999999 truncating down to 1 instead of rounding up to 2.
\end{DoxyParagraph}
To accomplish this behavior, the taper multiplies its unrounded parameter values by Real\+Precision, rounds the result to the nearest valid value, then divides Real\+Precision back out. ~\newline


Rounding will be disabled if Real\+Precision is set to a value less than 1
