\doxysubsection{Detailed Description}
Publicly inherits from I\+A\+C\+F\+Unknown.\+Publicly inherits from I\+A\+C\+F\+Unknown.\+Provides the abstract interface for component creation. 

\begin{DoxyRemark}{Remarks}
Every component implemenation class must have a corresponding class factory that implements the \mbox{\hyperlink{class_i_a_c_f_class_factory}{I\+A\+C\+F\+Class\+Factory}} interface.
\end{DoxyRemark}
For general information about class factories see the associated documentation for I\+Class\+Factory. \begin{DoxyNote}{Note}
Plug-\/ins implement this interface to encapsulate component creation \mbox{[}Abstract Factory\mbox{]}. The host will use instances of this interface returned from the plug-\/in\textquotesingle{}s A\+C\+F\+Get\+Class\+Factory callback function to manage the creation of new components. 
\end{DoxyNote}
