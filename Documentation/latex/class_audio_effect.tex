This process method must be provided. It takes input data, applies its pocessing algorithm, and then puts the result to the output by overwriting the output buffer.

\begin{DoxyWarning}{Warning}
Never call any Mac OS 9 functions (or other functions which call into the OS) inside your audio process function! This will crash the system when your plug-\/in is run in MP (multiprocessor) mode. If you must call into the OS, you must use M\+P\+Remote\+Call () (see Apples\textquotesingle{} documentation), or explicitly use functions which are documented by Apple to be MP safe. On Mac OS X read the system header files to be sure that you only call thread safe functions. 
\end{DoxyWarning}
