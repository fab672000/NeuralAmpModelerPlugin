\doxysubsection{Detailed Description}
Types for interacting with GL resources created externally to Skia. Gr\+Backend\+Objects for GL textures are really const Gr\+G\+L\+Texture$\ast$. The f\+Format here should be a sized, internal format for the texture. We will try to use the sized format if the GL Context supports it, otherwise we will internally fall back to using the base internal formats.
