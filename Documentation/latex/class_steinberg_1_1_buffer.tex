\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_steinberg_1_1_buffer}{Buffer}}.

A \mbox{\hyperlink{class_steinberg_1_1_buffer}{Buffer}} is an object-\/oriented wrapper for a piece of memory. It adds several utility functions, e.\+g. for managing the size of the \mbox{\hyperlink{class_steinberg_1_1_buffer}{Buffer}}, appending or prepending values or strings to it. Internally it uses the standard memory functions malloc(), free(), etc.
