\doxysubsection{Detailed Description}
The sole purpose of this namespace is to be able to import all functions and symbols that are expected to be used within operator() for indexing and slicing. If you already imported the whole \mbox{\hyperlink{namespace_eigen}{Eigen}} namespace\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using namespace }\mbox{\hyperlink{namespace_eigen}{Eigen}}; }
\end{DoxyCode}
 then you are already all set. Otherwise, if you don\textquotesingle{}t want/cannot import the whole \mbox{\hyperlink{namespace_eigen}{Eigen}} namespace, the following line\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using namespace }\mbox{\hyperlink{namespace_eigen_1_1indexing}{Eigen::indexing}}; }
\end{DoxyCode}
 is equivalent to\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::fix;}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::seq;}
\DoxyCodeLine{\textcolor{keyword}{using} \mbox{\hyperlink{namespace_eigen_a3a3c346d2a61d1e8e86e6fb4cf57fbda}{Eigen::seqN}};}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::placeholders::all;}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::placeholders::last;}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::placeholders::lastN;  \textcolor{comment}{// c++11 only}}
\DoxyCodeLine{\textcolor{keyword}{using} Eigen::placeholders::lastp1;}
\end{DoxyCode}
