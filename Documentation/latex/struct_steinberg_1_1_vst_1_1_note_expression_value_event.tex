\doxysubsection{Detailed Description}
Note Expression Value event. Used in \mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_event}{Event}} (union) A note expression event affects one single playing note (referring its note\+Id). This kind of event is send from host to the plug-\/in like other events (\mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_note_on_event}{Note\+On\+Event}}, \mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_note_off_event}{Note\+Off\+Event}},...) in \mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_process_data}{Process\+Data}} during the process call. Note expression events for a specific note\+Id can only occur after a \mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_note_on_event}{Note\+On\+Event}}. The host must take care that the event list (\mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_event_list}{I\+Event\+List}}) is properly sorted. Expression events are always absolute normalized values \mbox{[}0.\+0, 1.\+0\mbox{]}.
