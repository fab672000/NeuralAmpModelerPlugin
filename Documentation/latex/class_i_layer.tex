\doxysubsection{Detailed Description}
An abstraction that is used to store a temporary raster image/framebuffer. The layer is drawn with a specific offset to the graphics context. I\+Layers take ownership of the underlying bitmaps In G\+P\+U-\/based backends (Nano\+VG), this is a texture.
