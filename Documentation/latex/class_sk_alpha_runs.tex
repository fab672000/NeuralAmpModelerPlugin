\doxysubsection{Detailed Description}
Sparse array of run-\/length-\/encoded alpha (supersampling coverage) values. Sparseness allows us to independently compose several paths into the same \mbox{\hyperlink{class_sk_alpha_runs}{Sk\+Alpha\+Runs}} buffer.
