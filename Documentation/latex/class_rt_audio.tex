\doxysubsection{Detailed Description}
Realtime audio i/o C++ classes. 

\mbox{\hyperlink{class_rt_audio}{Rt\+Audio}} provides a common A\+PI (Application Programming Interface) for realtime audio input/output across Linux (native A\+L\+SA, Jack, and O\+SS), Macintosh OS X (Core\+Audio and Jack), and Windows (Direct\+Sound, A\+S\+IO and W\+A\+S\+A\+PI) operating systems.
