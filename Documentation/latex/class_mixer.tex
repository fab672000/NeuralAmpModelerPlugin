\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_mixer}{Mixer}} object which sums the output from multiple tracks into a single output. The number of input channels on each track must match the number of output channels (default 1=mono). This can be changed by calling {\ttfamily set\+Channel\+Count}. The inputs to the mixer are not owned by the mixer, they should not be deleted while rendering.
