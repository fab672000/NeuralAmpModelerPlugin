Compute useful matrices for use with sizing in underlying libraries.

There are two kinds of text size, a \textquotesingle{}requested/logical size\textquotesingle{} which is like asking for size \textquotesingle{}12\textquotesingle{} and a \textquotesingle{}real\textquotesingle{} size which is the size after the matrix is applied. The matrices produced by this method are based on the \textquotesingle{}real\textquotesingle{} size. This method effectively finds the total device matrix and decomposes it in various ways.

The most useful decomposition is into \textquotesingle{}scale\textquotesingle{} and \textquotesingle{}remaining\textquotesingle{}. The \textquotesingle{}scale\textquotesingle{} is applied first and then the \textquotesingle{}remaining\textquotesingle{} to fully apply the total matrix. This decomposition is useful when the text size (\textquotesingle{}scale\textquotesingle{}) may have meaning apart from the total matrix. This is true when hinting, and sometimes true for other properties as well.

The second (optional) decomposition is of \textquotesingle{}remaining\textquotesingle{} into a non-\/rotational part \textquotesingle{}remaining\+Without\+Rotation\textquotesingle{} and a rotational part \textquotesingle{}remaining\+Rotation\textquotesingle{}. The \textquotesingle{}scale\textquotesingle{} is applied first, then \textquotesingle{}remaining\+Without\+Rotation\textquotesingle{}, then \textquotesingle{}remaining\+Rotation\textquotesingle{} to fully apply the total matrix. This decomposition is helpful when only horizontal metrics can be trusted, so the \textquotesingle{}scale\textquotesingle{} and \textquotesingle{}remaining\+Without\+Rotation\textquotesingle{} will be handled by the underlying library, but the final rotation \textquotesingle{}remaining\+Rotation\textquotesingle{} will be handled manually.

The \textquotesingle{}total\textquotesingle{} matrix is also (optionally) available. This is useful in cases where the underlying library will not be used, often when working directly with font data.

The parameters \textquotesingle{}scale\textquotesingle{} and \textquotesingle{}remaining\textquotesingle{} are required, the other pointers may be nullptr.
