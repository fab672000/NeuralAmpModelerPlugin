\doxysubsection{Detailed Description}
A key used for scratch resources. There are three important rules about scratch keys\+:
\begin{DoxyItemize}
\item Multiple resources can share the same scratch key. Therefore resources assigned the same scratch key should be interchangeable with respect to the code that uses them.
\item A resource can have at most one scratch key and it is set at resource creation by the resource itself.
\item When a scratch resource is ref\textquotesingle{}ed it will not be returned from the cache for a subsequent cache request until all refs are released. This facilitates using a scratch key for multiple render-\/to-\/texture scenarios. An example is a separable blur\+:
\end{DoxyItemize}
