When a (instrument) plug-\/in supports such interface, the host could get from the plug-\/in the current set of used key switches (megatrig/articulation) for a given channel of a event bus and then automatically use them (like in Cubase 6) to create V\+ST Expression Map (allowing to associated symbol to a given articulation / key switch).
