\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classskottie_1_1_property_observer}{Property\+Observer}} can be used to track and manipulate certain properties of \char`\"{}interesting\char`\"{} Lottie nodes.

When registered with an animation builder, \mbox{\hyperlink{classskottie_1_1_property_observer}{Property\+Observer}} receives notifications for various properties of layer and shape nodes. The $\vert$node\+\_\+name$\vert$ argument corresponds to the name (\char`\"{}nm\char`\"{}) node property.
