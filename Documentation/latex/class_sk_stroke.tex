\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_stroke}{Sk\+Stroke}} is the utility class that constructs paths by stroking geometries (lines, rects, ovals, roundrects, paths). This is invoked when a geometry or text is drawn in a canvas with the k\+Stroke\+\_\+\+Mask bit set in the paint.
