\doxysubsection{Detailed Description}
Description interface for an effect\textquotesingle{}s (plug-\/in type\textquotesingle{}s) components. 

\textbackslash{}hostimp

Each Effect represents a different \char`\"{}type\char`\"{} of plug-\/in. The host will present different Effects to the user as separate products, even if they are derived from the same \mbox{\hyperlink{class_a_a_x___i_collection}{A\+A\+X\+\_\+\+I\+Collection}} description.
