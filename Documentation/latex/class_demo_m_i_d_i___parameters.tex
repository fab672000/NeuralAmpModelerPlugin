M\+I\+DI update callback. 

This method is called by the host for each pending M\+I\+DI packet for M\+I\+DI nodes in algorithm context structure. Overwrite this method in Plug-\/\+In\textquotesingle{}s Effect\+Parameter class if you want to receive M\+I\+DI data packets directly in the data model. M\+I\+DI data will also be delivered to the Effect algorithm.

The host calls this method in Effects that register one or more M\+I\+DI nodes using \mbox{\hyperlink{class_a_a_x___i_component_descriptor_a6284dda9ccca898e33075de29dad4e39}{A\+A\+X\+\_\+\+I\+Component\+Descriptor\+::\+Add\+M\+I\+D\+I\+Node()}}. Effects that do not require M\+I\+DI data to be sent to the plug-\/in algorithm should override \mbox{\hyperlink{class_a_a_x___i_a_c_f_effect_parameters___v2_ab4ec161f64086070083c21b566354861}{Update\+Control\+M\+I\+D\+I\+Nodes()}}.
