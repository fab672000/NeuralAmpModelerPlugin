\doxysubsection{Detailed Description}
The basic interface of all interfaces.


\begin{DoxyItemize}
\item The \mbox{\hyperlink{class_steinberg_1_1_f_unknown_a784e44c90535adbbf7f83ca670e195db}{F\+Unknown\+::query\+Interface}} method is used to retrieve pointers to other interfaces of the object.

If no more references exist, the object is destroyed in memory.
\end{DoxyItemize}

Interfaces are identified by 16 byte Globally Unique Identifiers. The S\+DK provides a class called \mbox{\hyperlink{class_steinberg_1_1_f_u_i_d}{F\+U\+ID}} for this purpose.
