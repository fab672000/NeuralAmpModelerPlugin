\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_u_text}{U\+Text}} struct. Provides the interface between the generic \mbox{\hyperlink{struct_u_text}{U\+Text}} access code and the \mbox{\hyperlink{struct_u_text}{U\+Text}} provider code that works on specific kinds of text (U\+T\+F-\/8, noncontiguous U\+T\+F-\/16, whatever.)

Applications that are using predefined types of text providers to pass text data to I\+CU services will have no need to view the internals of the \mbox{\hyperlink{struct_u_text}{U\+Text}} structs that they open.
