Helper Class wrapping a V\+ST 3 plug-\/in to a Audio Unit v3 plug-\/in\hypertarget{_a_uv3_wrapper_AUv3Introduction}{}\doxysection{Introduction}\label{_a_uv3_wrapper_AUv3Introduction}
The V\+ST 3 S\+DK comes with a helper class which wraps one V\+ST 3 Audio Processor and Edit Controller to a A\+Uv3 plug-\/in.

The wrapped Audio\+Unit does support M\+PE when the V\+S\+T3 plug-\/in has Note Expression support. You need to implement \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_note_expression_physical_u_i_mapping}{Steinberg\+::\+Vst\+::\+I\+Note\+Expression\+Physical\+U\+I\+Mapping}} to map your Note Expression to the limited three expressions defined by M\+PE. ~\newline
