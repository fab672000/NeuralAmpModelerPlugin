\doxysubsection{Detailed Description}
Publicly inherits from I\+A\+C\+F\+Definition.\+Provides the abstract interface for defining A\+CF plug-\/ins. 

\begin{DoxyRemark}{Remarks}
An object that implements this interface must be returned to the host from the A\+C\+F\+Register\+Plugin callback. This object is only valid during the registration of the plug-\/in. 
\end{DoxyRemark}
\begin{DoxyNote}{Note}
This interface is implemented by the host. Plug-\/in The plug-\/in will use this interface to register itself with the A\+CF host. 

The components implemented by a plug-\/in are registered separately in the A\+C\+F\+Register\+Component callback. 
\end{DoxyNote}
