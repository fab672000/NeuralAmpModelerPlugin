bounds defines an optional bounding box, used to suppress drawing when \mbox{\hyperlink{class_sk_text_blob}{Sk\+Text\+Blob}} bounds does not intersect \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}} bounds. If bounds is nullptr, \mbox{\hyperlink{class_sk_text_blob}{Sk\+Text\+Blob}} bounds is computed from y, \mbox{\hyperlink{struct_sk_text_blob_builder_1_1_run_buffer_a69dc3694061e726781687c4bd98f3fa5}{Run\+Buffer\+::pos}}, and \mbox{\hyperlink{struct_sk_text_blob_builder_1_1_run_buffer_a81d605d234e2eed9399779e78efa98f0}{Run\+Buffer\+::glyphs}} metrics.
