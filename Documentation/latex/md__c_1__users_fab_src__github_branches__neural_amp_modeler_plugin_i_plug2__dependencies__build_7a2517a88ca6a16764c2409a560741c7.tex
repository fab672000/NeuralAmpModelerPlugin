(Oboe) {\bfseries{iolib}} contains classes implementing streaming audio playback and mixing from multiple sources. It\textquotesingle{}s purpose is to demonstrate best practices and provide reusable code.

Note\+: the more general name of \char`\"{}iolib\char`\"{} was chosen since it is presumed that this project will eventually implement audio capture capability.

{\bfseries{iolib}} is written in C++ and is intended to be called from Android native code.
