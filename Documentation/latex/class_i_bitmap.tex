\doxysubsection{Detailed Description}
User-\/facing bitmap abstraction that you use to manage bitmap data, independant of draw class/platform. \mbox{\hyperlink{class_i_bitmap}{I\+Bitmap}} doesn\textquotesingle{}t actually own the image data \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_a_p_i_bitmap}{A\+P\+I\+Bitmap}} An \mbox{\hyperlink{class_i_bitmap}{I\+Bitmap}}\textquotesingle{}s width and height are always in relation to a 1\+:1 (low dpi) screen. Any scaling happens at the drawing stage. 
\end{DoxySeeAlso}
