\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_path_effect}{Sk\+Path\+Effect}} is the base class for objects in the \mbox{\hyperlink{class_sk_paint}{Sk\+Paint}} that affect the geometry of a drawing primitive before it is transformed by the canvas\textquotesingle{} matrix and drawn.

Dashing is implemented as a subclass of \mbox{\hyperlink{class_sk_path_effect}{Sk\+Path\+Effect}}.

Given a src path (input) and a stroke-\/rec (input and output), apply this effect to the src path, returning the new path in dst, and return true. If this effect cannot be applied, return false and ignore dst and stroke-\/rec.

The stroke-\/rec specifies the initial request for stroking (if any). The effect can treat this as input only, or it can choose to change the rec as well. For example, the effect can decide to change the stroke\textquotesingle{}s width or join, or the effect can change the rec from stroke to fill (or fill to stroke) in addition to returning a new (dst) path.

If this method returns true, the caller will apply (as needed) the resulting stroke-\/rec to dst and then draw. \mbox{\Hypertarget{class_sk_path_effect_a8188426f71783ffd0e9d16bfec4e0512}\label{class_sk_path_effect_a8188426f71783ffd0e9d16bfec4e0512}}
