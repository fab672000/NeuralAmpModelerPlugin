struct Eigen\+::\+Matrix\+Exponential\+Return\+Value$<$ Derived $>$}

Proxy for the matrix exponential of some matrix (expression). 


\begin{DoxyTemplParams}{Template Parameters}
{\em Derived} & Type of the argument to the matrix exponential.\\
\hline
\end{DoxyTemplParams}
This class holds the argument to the matrix exponential until it is assigned or evaluated for some other reason (so the argument should not be changed in the meantime). It is the return type of \mbox{\hyperlink{group__core__func__exponential_ga6e9557409b21768860d8f7d974bb41dc}{Matrix\+Base\+::exp()}} and most of the time this is the only way it is used.
