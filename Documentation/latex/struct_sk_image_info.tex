\doxysubsection{Detailed Description}
Describes pixel dimensions and encoding. \mbox{\hyperlink{class_sk_bitmap}{Sk\+Bitmap}}, \mbox{\hyperlink{class_sk_image}{Sk\+Image}}, Pix\+Map, and \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}} can be created from \mbox{\hyperlink{struct_sk_image_info}{Sk\+Image\+Info}}. \mbox{\hyperlink{struct_sk_image_info}{Sk\+Image\+Info}} can be retrieved from \mbox{\hyperlink{class_sk_bitmap}{Sk\+Bitmap}} and \mbox{\hyperlink{class_sk_pixmap}{Sk\+Pixmap}}, but not from \mbox{\hyperlink{class_sk_image}{Sk\+Image}} and \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}}. For example, \mbox{\hyperlink{class_sk_image}{Sk\+Image}} and \mbox{\hyperlink{class_sk_surface}{Sk\+Surface}} implementations may defer pixel depth, so may not completely specify \mbox{\hyperlink{struct_sk_image_info}{Sk\+Image\+Info}}.

\mbox{\hyperlink{struct_sk_image_info}{Sk\+Image\+Info}} contains dimensions, the pixel integral width and height. It encodes how pixel bits describe alpha, transparency; color components red, blue, and green; and \mbox{\hyperlink{class_sk_color_space}{Sk\+Color\+Space}}, the range and linearity of colors.
