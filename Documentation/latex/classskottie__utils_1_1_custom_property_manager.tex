\doxysubsection{Detailed Description}
\mbox{\hyperlink{classskottie__utils_1_1_custom_property_manager}{Custom\+Property\+Manager}} implements a property management scheme where color/opacity/transform attributes are grouped and manipulated by name (one-\/to-\/many mapping).


\begin{DoxyItemize}
\item setters apply the value to all properties in a named group
\item getters return all the managed property groups, and the first value within each of them (unchecked assumption\+: all properties within the same group have the same value)
\end{DoxyItemize}

Attach to an Animation\+::\+Builder using the utility methods below to intercept properties and markers at build time.
