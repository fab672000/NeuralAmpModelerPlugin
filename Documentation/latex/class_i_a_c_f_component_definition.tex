\doxysubsection{Detailed Description}
Publicly inherits from I\+A\+C\+F\+Definition.\+Provides the abstract interface for defining generic components. 

\begin{DoxyRemark}{Remarks}
This interface is used to define the basic attributes required by a generic component. This interface is provided as the basis for plug-\/ins to define their own sharable components. An object that implements this interface must be returned to the host from the A\+C\+F\+Register\+Component callback. This object is only valid during the registration of the plug-\/in.
\end{DoxyRemark}
\begin{DoxyNote}{Note}
The A\+CF may use component subclass interface for each component type
\begin{DoxyItemize}
\item image effects
\item parameters
\item meta sync
\item etc...
\end{DoxyItemize}

This interface is implemented by the host. The plug-\/in will use this interface to define a generic component. 
\end{DoxyNote}
