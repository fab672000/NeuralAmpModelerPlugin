\doxysubsection{Detailed Description}
Used to manage a rectangular area, independent of draw class/platform. An \mbox{\hyperlink{struct_i_r_e_c_t}{I\+R\+E\+CT}} is always specified in 1\+:1 pixels, any scaling for high D\+PI happens in the drawing class. In \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} 0,0 is top left.
