{\footnotesize\ttfamily bool Sk\+Anim\+Codec\+Player\+::seek (\begin{DoxyParamCaption}\item[{uint32\+\_\+t}]{msec }\end{DoxyParamCaption})}

Finds the closest frame associated with the time code (in milliseconds) and sets that to be the current frame (call \mbox{\hyperlink{class_sk_anim_codec_player_a0a1b98e18a875ee7d84c91ab3cf6b0a1}{get\+Frame()}} to retrieve that image). Returns true iff this call to \mbox{\hyperlink{class_sk_anim_codec_player_adfc361eb8e2085948aa47ef38493e532}{seek()}} changed the \char`\"{}current frame\char`\"{} for the animation. Thus if \mbox{\hyperlink{class_sk_anim_codec_player_adfc361eb8e2085948aa47ef38493e532}{seek()}} returns false, then \mbox{\hyperlink{class_sk_anim_codec_player_a0a1b98e18a875ee7d84c91ab3cf6b0a1}{get\+Frame()}} will return the same image as it did before this call to \mbox{\hyperlink{class_sk_anim_codec_player_adfc361eb8e2085948aa47ef38493e532}{seek()}}.
