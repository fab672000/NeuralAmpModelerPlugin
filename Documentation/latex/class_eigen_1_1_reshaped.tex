\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Xpr\+Type, int Rows, int Cols, int Order$>$\newline
class Eigen\+::\+Reshaped$<$ Xpr\+Type, Rows, Cols, Order $>$}

Expression of a fixed-\/size or dynamic-\/size reshape. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Xpr\+Type} & the type of the expression in which we are taking a reshape \\
\hline
{\em Rows} & the number of rows of the reshape we are taking at compile time (optional) \\
\hline
{\em Cols} & the number of columns of the reshape we are taking at compile time (optional) \\
\hline
{\em Order} & can be Col\+Major or Row\+Major, default is Col\+Major.\\
\hline
\end{DoxyTemplParams}
This class represents an expression of either a fixed-\/size or dynamic-\/size reshape. It is the return type of Dense\+Base\+::reshaped(\+N\+Rows\+Type,\+N\+Cols\+Type) and most of the time this is the only way it is used.

If you want to directly manipulate reshaped expressions, for instance if you want to write a function returning such an expression, it is advised to use the {\itshape auto} keyword for such use cases.
