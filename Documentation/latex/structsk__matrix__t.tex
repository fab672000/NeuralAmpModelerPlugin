\doxysubsection{Detailed Description}
The \mbox{\hyperlink{structsk__matrix__t}{sk\+\_\+matrix\+\_\+t}} struct holds a 3x3 perspective matrix for transforming coordinates\+: \begin{DoxyVerb}(X,Y) = T[M]((x,y))
X = (M[0] * x + M[1] * y + M[2]) / (M[6] * x + M[7] * y + M[8]);
Y = (M[3] * x + M[4] * y + M[5]) / (M[6] * x + M[7] * y + M[8]);
\end{DoxyVerb}


Therefore, the identity matrix is \begin{DoxyVerb}sk_matrix_t identity = {{1, 0, 0,
                         0, 1, 0,
                         0, 0, 1}};
\end{DoxyVerb}


A matrix that scales by sx and sy is\+: \begin{DoxyVerb}sk_matrix_t scale = {{sx, 0,  0,
                      0,  sy, 0,
                      0,  0,  1}};
\end{DoxyVerb}


A matrix that translates by tx and ty is\+: \begin{DoxyVerb}sk_matrix_t translate = {{1, 0, tx,
                          0, 1, ty,
                          0, 0, 1}};
\end{DoxyVerb}


A matrix that rotates around the origin by A radians\+: \begin{DoxyVerb}sk_matrix_t rotate = {{cos(A), -sin(A), 0,
