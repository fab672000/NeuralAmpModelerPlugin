\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_sk_typeface}{Sk\+Typeface}} class specifies the typeface and intrinsic style of a font. This is used in the paint, along with optionally algorithmic settings like text\+Size, text\+SkewX, text\+ScaleX, k\+Fake\+Bold\+Text\+\_\+\+Mask, to specify how text appears when drawn (and measured).

Typeface objects are immutable, and so they can be shared between threads.
