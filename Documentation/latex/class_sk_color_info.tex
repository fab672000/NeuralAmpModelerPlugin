\doxysubsection{Detailed Description}
Describes pixel and encoding. \mbox{\hyperlink{struct_sk_image_info}{Sk\+Image\+Info}} can be created from \mbox{\hyperlink{class_sk_color_info}{Sk\+Color\+Info}} by providing dimensions.

It encodes how pixel bits describe alpha, transparency; color components red, blue, and green; and \mbox{\hyperlink{class_sk_color_space}{Sk\+Color\+Space}}, the range and linearity of colors.
