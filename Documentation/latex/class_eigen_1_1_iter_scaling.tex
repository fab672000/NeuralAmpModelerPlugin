\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\+Type\+\_\+$>$\newline
class Eigen\+::\+Iter\+Scaling$<$ Matrix\+Type\+\_\+ $>$}

iterative scaling algorithm to equilibrate rows and column norms in matrices 

This class can be used as a preprocessing tool to accelerate the convergence of iterative methods

This feature is useful to limit the pivoting amount during L\+U/\+I\+LU factorization The scaling strategy as presented here preserves the symmetry of the problem N\+O\+TE It is assumed that the matrix does not have empty row or column,

Example with key steps 
\begin{DoxyCode}{0}
\DoxyCodeLine{VectorXd x(n), b(n);}
\DoxyCodeLine{SparseMatrix<double> A;}
\DoxyCodeLine{\textcolor{comment}{// fill A and b;}}
\DoxyCodeLine{IterScaling<SparseMatrix<double> > scal; }
\DoxyCodeLine{\textcolor{comment}{// Compute the left and right scaling vectors. The matrix is equilibrated at output}}
\DoxyCodeLine{scal.computeRef(A); }
\DoxyCodeLine{\textcolor{comment}{// Scale the right hand side}}
\DoxyCodeLine{b = scal.LeftScaling().cwiseProduct(b); }
\DoxyCodeLine{\textcolor{comment}{// Now, solve the equilibrated linear system with any available solver}}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// Scale back the computed solution}}
\DoxyCodeLine{x = scal.RightScaling().cwiseProduct(x); }
\end{DoxyCode}
