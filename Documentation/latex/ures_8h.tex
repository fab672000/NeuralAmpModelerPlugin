\doxysubsection{Detailed Description}
C A\+PI\+: Resource Bundle. 

\doxysubsubsection*{C A\+PI\+: Resource Bundle}

C A\+PI representing a collection of resource information pertaining to a given locale. A resource bundle provides a way of accessing locale-\/ specific information in a data file. You create a resource bundle that manages the resources for a given locale and then ask it for individual resources. 

Resource bundles in I\+C\+U4C are currently defined using text files which conform to the following \href{http://source.icu-project.org/repos/icu/icuhtml/trunk/design/bnf_rb.txt}{\texttt{ B\+NF definition}}. More on resource bundle concepts and syntax can be found in the \href{http://icu-project.org/userguide/ResourceManagement.html}{\texttt{ Users Guide}}.
