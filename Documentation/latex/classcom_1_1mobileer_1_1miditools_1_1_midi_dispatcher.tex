\doxysubsection{Detailed Description}
Utility class for dispatching M\+I\+DI data to a list of \mbox{\hyperlink{}{android.\+media.\+midi.\+Midi\+Receiver}}s. This class subclasses \mbox{\hyperlink{}{android.\+media.\+midi.\+Midi\+Receiver}} and dispatches any data it receives to its receiver list. Any receivers that throw an exception upon receiving data will be automatically removed from the receiver list, but no I\+O\+Exception will be returned from the dispatcher\textquotesingle{}s \mbox{\hyperlink{}{android.\+media.\+midi.\+Midi\+Receiver\#on\+Receive}} in that case.
