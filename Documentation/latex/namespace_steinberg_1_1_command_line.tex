\doxysubsection{Detailed Description}
Very simple command-\/line parser.

Parses the command-\/line into a \mbox{\hyperlink{class_steinberg_1_1_command_line_1_1_variables_map}{Command\+Line\+::\+Variables\+Map}}.~\newline
The command-\/line parser uses \mbox{\hyperlink{class_steinberg_1_1_command_line_1_1_descriptions}{Command\+Line\+::\+Descriptions}} to define the available options.
