\doxysubsection{Detailed Description}
Handles the interaction between pointers and vector, matrix types. 

This extension defines an overloaded function, glm\+::value\+\_\+ptr, which takes any of the \mbox{\hyperlink{group__core__template}{core template types}}. It returns a pointer to the memory layout of the object. Matrix types store their values in column-\/major order.

This is useful for uploading data to matrices or copying data to buffer objects.
