\mbox{\hyperlink{class_sk_t_mask_pre_blend}{Sk\+T\+Mask\+Pre\+Blend}} is a tear-\/off of \mbox{\hyperlink{class_sk_t_mask_gamma}{Sk\+T\+Mask\+Gamma}}. It provides the tables to convert a linear alpha value for a given channel to a gamma correcting alpha value for that channel. This class is immutable.

If fR, fG, or fB is nullptr, all of them will be. This indicates that no mask pre blend should be applied. \mbox{\hyperlink{class_sk_t_mask_pre_blend_acf734819993e6860b6ae19297799b6c3}{Sk\+T\+Mask\+Pre\+Blend\+::is\+Applicable()}} is provided as a convenience function to test for the absence of this case.
