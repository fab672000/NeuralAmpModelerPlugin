\doxysubsection{Detailed Description}
Class definition for objects that help setting boolean variables. A stack allocated object of this type automatically sets an at construction time passed boolean variable to T\+R\+UE if the given condition is met. At the end of its own scope the stack object will reset the same boolean variable to F\+A\+L\+SE, if it wasn\textquotesingle{}t set so already. ~\newline
