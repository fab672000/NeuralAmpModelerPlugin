\doxysubsection{Detailed Description}
Superclass for all R\+I\+FF chunks. Handles the chunk ID and chunk size. Concrete subclasses include chunks for \textquotesingle{}R\+I\+FF\textquotesingle{} and \textquotesingle{}fmt \textquotesingle{} chunks.
