\doxysubsection{Detailed Description}
a base class for objects accepting O\+SC messages 

\mbox{\hyperlink{classoscfaust_1_1_message}{Message}} driven objects are hierarchically organized in a tree. They provides the necessary to dispatch an O\+SC message to its destination node, according to the message O\+SC address.

The principle of the dispatch is the following\+:
\begin{DoxyItemize}
\item first the \mbox{\hyperlink{classoscfaust_1_1_message_driven_af69d3cd75650dff6f2822206f3aa0ad4}{process\+Message()}} method should be called on the top level node
\item next process\+Message call propose 
\end{DoxyItemize}
