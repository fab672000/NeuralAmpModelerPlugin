\doxysubsection{Detailed Description}
This is a restricted form of \mbox{\hyperlink{class_sk_image}{Sk\+Image}} solely intended for internal use. It differs from \mbox{\hyperlink{class_sk_image}{Sk\+Image}} in that\+:
\begin{DoxyItemize}
\item it can only be backed by raster or gpu (no generators)
\item it can be backed by a Gr\+Texture\+Proxy larger than its nominal bounds
\item it can\textquotesingle{}t be drawn tiled
\item it can\textquotesingle{}t be drawn with M\+I\+P\+M\+A\+Ps It is similar to \mbox{\hyperlink{class_sk_image}{Sk\+Image}} in that it abstracts how the pixels are stored/represented.
\end{DoxyItemize}

Note\+: the contents of the backing storage outside of the subset rect are undefined.
