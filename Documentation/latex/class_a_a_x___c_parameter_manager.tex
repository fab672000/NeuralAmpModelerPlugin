\doxysubsection{Detailed Description}
A container object for plug-\/in parameters. 

This implementation uses a S\+TL vector to store a plug-\/in\textquotesingle{}s set of parameters. This class contains a real implementation of the A\+A\+X\+Library\+Features\+\_\+\+Parameter\+Manager (as opposed to a proxy.)

For more information, see A\+A\+X\+Library\+Features\+\_\+\+Parameter\+Manager.
