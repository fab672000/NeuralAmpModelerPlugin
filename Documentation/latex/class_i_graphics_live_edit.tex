\doxysubsection{Detailed Description}
A control to enable live modification of control layout in an \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} context in debug builds This is based on the work of Youlean, who first included it in i\+Plug-\/\+Youlean The lives outside the main \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} control stack and it can be added with \mbox{\hyperlink{class_i_graphics_a637237470659702554f53061b91c8d86}{I\+Graphics\+::\+Enable\+Live\+Edit()}}. It should not be used in the main control stack.
