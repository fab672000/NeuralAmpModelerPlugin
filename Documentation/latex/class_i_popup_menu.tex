\doxysubsection{Detailed Description}
A class for setting the contents of a pop up menu. 

An \mbox{\hyperlink{class_i_popup_menu}{I\+Popup\+Menu}} must not be declared as a temporary. In order for a receiving \mbox{\hyperlink{class_i_control}{I\+Control}} or lambda function to be triggered when something is selected, the menu should persist across function calls, therefore it should almost always be a member variable. An \mbox{\hyperlink{class_i_popup_menu}{I\+Popup\+Menu}} owns its sub items, including submenus This (and the platform implementations) are largely based on the V\+S\+T\+G\+UI C\+Option\+Menu
