\doxysubsection{Detailed Description}
We have two techniques for capturing the output of the supersampler\+:
\begin{DoxyItemize}
\item S\+U\+P\+E\+R\+M\+A\+SK, which records a large mask-\/bitmap this is often faster for small, complex objects
\item R\+LE, which records a rle-\/encoded scanline this is often faster for large objects with big spans
\end{DoxyItemize}

These blitters use two coordinate systems\+:
\begin{DoxyItemize}
\item destination coordinates, scale equal to the output -\/ often abbreviated with \textquotesingle{}i\textquotesingle{} or \textquotesingle{}\mbox{\hyperlink{struct_i}{I}}\textquotesingle{} in variable names
\item supersampled coordinates, scale equal to the output $\ast$ S\+C\+A\+LE 
\end{DoxyItemize}
