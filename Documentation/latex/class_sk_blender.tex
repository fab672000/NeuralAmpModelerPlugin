\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_blender}{Sk\+Blender}} represents a custom blend function in the Skia pipeline. When an \mbox{\hyperlink{class_sk_blender}{Sk\+Blender}} is present in a paint, the Sk\+Blend\+Mode is ignored. A blender combines a source color (the result of our paint) and destination color (from the canvas) into a final color.
