\doxysubsection{Detailed Description}
This is a helper for draw\+Vertices(). It is used to iterate over the triangles that are to be rendered based on an Sk\+Canvas\+::\+Vertex\+Mode and (optionally) an index array. It does not copy the index array and the client must ensure it remains valid for the lifetime of the \mbox{\hyperlink{struct_vert_state}{Vert\+State}} object.
