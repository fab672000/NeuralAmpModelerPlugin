\mbox{\hyperlink{class_block_index_iterator}{Block\+Index\+Iterator}} provides a reusable iterator template for collections built on top of a \mbox{\hyperlink{class_sk_block_allocator}{Sk\+Block\+Allocator}}, where each item is of the same type, and the index to an item can be iterated over in a known manner. It supports const and non-\/const, and forward and reverse, assuming it\textquotesingle{}s provided with proper functions for starting, ending, and advancing.
