\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_oscillator}{Oscillator}} that generates a sawtooth wave at the specified frequency and amplitude.

The waveform is not band-\/limited so it will have aliasing artifacts at higher frequencies.
