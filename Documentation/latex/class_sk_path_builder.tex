Arc is contained by tangent from last \mbox{\hyperlink{class_sk_path}{Sk\+Path}} point to p1, and tangent from p1 to p2. Arc is part of circle sized to radius, positioned so it touches both tangent lines.

Appends arc to \mbox{\hyperlink{class_sk_path}{Sk\+Path}}. Arc is implemented by one or more conic weighted to describe part of oval with radii (r.\+fX, r.\+fY) rotated by x\+Axis\+Rotate degrees. Arc curves from last \mbox{\hyperlink{class_sk_path}{Sk\+Path}} \mbox{\hyperlink{struct_sk_point}{Sk\+Point}} to (xy.\+fX, xy.\+fY), choosing one of four possible routes\+: clockwise or counterclockwise, and smaller or larger.

Arc sweep is always less than 360 degrees. \mbox{\hyperlink{class_sk_path_builder_a111a460f4fc0bb4d86cd6a191f1fe117}{arc\+To()}} appends line to xy if either radii are zero, or if last \mbox{\hyperlink{class_sk_path}{Sk\+Path}} \mbox{\hyperlink{struct_sk_point}{Sk\+Point}} equals (xy.\+fX, xy.\+fY). \mbox{\hyperlink{class_sk_path_builder_a111a460f4fc0bb4d86cd6a191f1fe117}{arc\+To()}} scales radii r to fit last \mbox{\hyperlink{class_sk_path}{Sk\+Path}} \mbox{\hyperlink{struct_sk_point}{Sk\+Point}} and xy if both are greater than zero but too small to describe an arc.

\mbox{\hyperlink{class_sk_path_builder_a111a460f4fc0bb4d86cd6a191f1fe117}{arc\+To()}} appends up to four conic curves. \mbox{\hyperlink{class_sk_path_builder_a111a460f4fc0bb4d86cd6a191f1fe117}{arc\+To()}} implements the functionality of S\+VG arc, although S\+VG sweep-\/flag value is opposite the integer value of sweep; S\+VG sweep-\/flag uses 1 for clockwise, while k\+C\+W\+\_\+\+Direction cast to int is zero.
