\doxysubsection{Detailed Description}
Describes a Gr\+Color\+Type by how many bits are used for each color component and how they are encoded. Currently all the non-\/zero channels share a single Gr\+Color\+Type\+Encoding. This could be expanded to store separate encodings and to indicate which bits belong to which components.
