Queries the host for the current \mbox{\hyperlink{_a_a_x___enums_8h_a47756e0a56d00468b7045eb26500cb78}{modifier keys}}. 

This method returns a bit mask with bits set for each of the currently active modifier keys. This method does not return the state of the \mbox{\hyperlink{_a_a_x___enums_8h_a47756e0a56d00468b7045eb26500cb78a6d7914ad68992fa2670fc0327704cbb7}{A\+A\+X\+\_\+e\+Modifiers\+\_\+\+Secondary\+Button}}.

\textbackslash{}compatibility Although this method allows plug-\/ins to acquire the current state of the Windows key (normally blocked by Pro Tools), plug-\/ins should not use key combinations that require this key.
