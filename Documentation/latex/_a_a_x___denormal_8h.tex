Clamps very small floating point values to zero. 

On Pentiums and Pentium I\+Is the generation of denormal floats causes enormous performance losses. This routine removes denormals by clamping very small values to zero. The clamping threshold is very small, but is not the absolute minimum. If absolute minimum clamping is desired, use \mbox{\hyperlink{_a_a_x___denormal_8h_ac52b6214a1de32e59b90ddbc1878e515}{A\+A\+X\+::\+De\+Denormal\+Fine()}} \mbox{\Hypertarget{_a_a_x___denormal_8h_aedfd1762f559291ceddd53f8ce8862a5}\label{_a_a_x___denormal_8h_aedfd1762f559291ceddd53f8ce8862a5}}
