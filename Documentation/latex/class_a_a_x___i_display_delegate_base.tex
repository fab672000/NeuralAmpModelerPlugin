\doxysubsection{Detailed Description}
Defines the display behavior for a parameter. 

This interface represents a delegate class to be used in conjunction with \mbox{\hyperlink{class_a_a_x___i_parameter}{A\+A\+X\+\_\+\+I\+Parameter}}. \mbox{\hyperlink{class_a_a_x___i_parameter}{A\+A\+X\+\_\+\+I\+Parameter}} delegates all conversion operations between strings and real parameter values to classes that meet this interface. You can think of \mbox{\hyperlink{class_a_a_x___i_taper_delegate}{A\+A\+X\+\_\+\+I\+Taper\+Delegate}} subclasses as simple string serialization routines that enable a specific string conversions for an arbitrary parameter.

For more information about how parameter delegates operate, see the \mbox{\hyperlink{class_a_a_x___i_taper_delegate}{A\+A\+X\+\_\+\+I\+Taper\+Delegate}} and A\+A\+X\+Library\+Features\+\_\+\+Parameter\+Manager documentation.
