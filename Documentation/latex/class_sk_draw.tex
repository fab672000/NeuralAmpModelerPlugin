Helper function that creates a mask from a path and an optional maskfilter. Note however, that the resulting mask will not have been actually filtered, that must be done afterwards (by calling filter\+Mask). The maskfilter is provided solely to assist in computing the mask\textquotesingle{}s bounds (if the mode requests that).
