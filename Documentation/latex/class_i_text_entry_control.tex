\doxysubsection{Detailed Description}
A Text entry widget drawn by \mbox{\hyperlink{class_i_graphics}{I\+Graphics}}. This is a special control that lives outside the main \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} control stack. It can be added with \mbox{\hyperlink{class_i_graphics_a4f57da0dc3b88c6b7c4f7d18b14b3d71}{I\+Graphics\+::\+Attach\+Text\+Entry\+Control()}}. It should not be used in the main control stack.
