\doxysubsection{Detailed Description}
Light weight class for reading data from a memory block. The R\+Buffer is given the buffer to read from, with either a specified size or no size (in which case no range checking is performed). It is iillegal to attempt to read a value from an empty R\+Buffer (data == null).
