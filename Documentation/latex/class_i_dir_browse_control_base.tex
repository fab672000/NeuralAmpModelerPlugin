\doxysubsection{Detailed Description}
An abstract \mbox{\hyperlink{class_i_control}{I\+Control}} base class that you can inherit from in order to make a control that pops up a menu to browse files Optionally with a specific extension. Add paths which will appear as subfolders at the root level menu. If only one path is added there will be no submenu. When you call \mbox{\hyperlink{class_i_dir_browse_control_base_abfcd3943d771dee725acd81b0fffda52}{Setup\+Menu()}} the added paths are scanned and any matching files in those paths are added to the menu.
