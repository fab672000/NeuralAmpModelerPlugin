\doxysubsection{Detailed Description}
Color\+Filters are optional objects in the drawing pipeline. When present in a paint, they are called with the \char`\"{}src\char`\"{} colors, and return new colors, which are then passed onto the next stage (either Image\+Filter or Xfermode).

All subclasses are required to be reentrant-\/safe \+: it must be legal to share the same instance between several threads.
