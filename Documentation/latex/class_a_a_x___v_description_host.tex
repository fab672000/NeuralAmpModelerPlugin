\doxysubsection{Detailed Description}
Versioned wrapper for access to host service interfaces provided during plug-\/in description

This object aggregates access to \mbox{\hyperlink{class_a_a_x___i_a_c_f_description_host}{A\+A\+X\+\_\+\+I\+A\+C\+F\+Description\+Host}} and \mbox{\hyperlink{class_i_a_c_f_definition}{I\+A\+C\+F\+Definition}}, with support depending on the interface support level of the \mbox{\hyperlink{interface_i_a_c_f_unknown}{I\+A\+C\+F\+Unknown}} which is passed to this object upon creation.
