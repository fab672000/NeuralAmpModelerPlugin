{\bfseries{Drum\+Thumper}} (apolgies to \href{https://www.youtube.com/watch?v=2H5uWRjFsGc}{\texttt{ Chumbawamba}}) is a \char`\"{}\+Drum Pad\char`\"{} app which demonstrates best-\/practices for low-\/latency audio playback using the Android {\bfseries{Oboe}} A\+PI. {\bfseries{Drum\+Thumper}} consists of a set of trigger pad widgets and an optional \mbox{\hyperlink{class_u_i}{UI}} for controlling the level and stereo placement of each of the virtual drums. The audio samples are stored in application resources as W\+AV data. This is parsed and loaded (by routines in {\bfseries{parselib}}) into memory blocks. The audio samples are mixed and played by routines in {\bfseries{iolib}}.
