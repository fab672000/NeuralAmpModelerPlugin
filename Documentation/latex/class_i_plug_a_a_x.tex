If {\itshape chunk\+ID} is one of the plug-\/in\textquotesingle{}s custom chunks, initialize {\itshape $\ast$size} to the size of the chunk\textquotesingle{}s data in bytes.

This method is invoked every time a chunk is saved, therefore it is possible to have dynamically sized chunks. However, note that each call to \mbox{\hyperlink{class_i_plug_a_a_x_ad7619f5d7a5963cf1f857765bc7d1861}{Get\+Chunk\+Size()}} will correspond to a following call to \mbox{\hyperlink{class_i_plug_a_a_x_a5139e084a69f5c9cde254fac38bc4aec}{Get\+Chunk()}}. \textbackslash{}legacy In A\+AX, the value provided by \mbox{\hyperlink{class_i_plug_a_a_x_ad7619f5d7a5963cf1f857765bc7d1861}{Get\+Chunk\+Size()}} should {\itshape N\+OT} include the size of the chunk header.
