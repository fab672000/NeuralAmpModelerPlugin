\doxysubsection{Detailed Description}
A base control for a pop-\/up menu/drop-\/down list that stays within the bounds of the \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} context. This is mainly used as a special control that lives outside the main \mbox{\hyperlink{class_i_graphics}{I\+Graphics}} control stack. For replacing generic menus this can be added with I\+Graphics\+::\+Attach\+Popup\+Menu(). If used in the main \mbox{\hyperlink{class_i_control}{I\+Control}} stack, you probably want it to be the very last control that is added, so that it gets drawn on top.
