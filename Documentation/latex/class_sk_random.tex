\doxysubsection{Detailed Description}
Utility class that implements pseudo random 32bit numbers using Marsaglia\textquotesingle{}s multiply-\/with-\/carry \char`\"{}mother of all\char`\"{} algorithm. Unlike rand(), this class holds its own state, so that multiple instances can be used with no side-\/effects.

Has a large period and all bits are well-\/randomized.
