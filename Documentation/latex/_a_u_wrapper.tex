The V\+ST 3 S\+DK comes with an Audio\+Unit wrapper, which wraps one V\+ST 3 Audio Processor and Edit Controller as an Audio\+Unit effect/instrument.

The wrapper is a small dynamic library which loads the {\bfseries{V\+ST 3 plug-\/in}}. No extra code is needed to be written. ~\newline
