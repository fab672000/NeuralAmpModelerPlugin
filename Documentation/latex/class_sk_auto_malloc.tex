\doxysubsection{Detailed Description}
Manage an allocated block of heap memory. This object is the sole manager of the lifetime of the block, so the caller must not call sk\+\_\+free() or delete on the block, unless \mbox{\hyperlink{class_sk_auto_malloc_a8a836f2b13a047e3209552494a8d96a0}{release()}} was called.
