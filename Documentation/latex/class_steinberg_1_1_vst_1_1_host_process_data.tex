\doxysubsection{Detailed Description}
Extension of \mbox{\hyperlink{struct_steinberg_1_1_vst_1_1_process_data}{Process\+Data}}.

Helps setting up the buffers for the process data structure for a component. When the prepare method is called with buffer\+Samples != 0 the buffer management is handled by this class. Otherwise the buffers need to be setup explicitly.
