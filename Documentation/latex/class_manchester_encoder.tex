\doxysubsection{Detailed Description}
Encode bytes using Manchester Coding scheme.

Manchester Code is self clocking. There is a transition in the middle of every bit. Zero is high then low. One is low then high.

This avoids having long DC sections that would droop when passed though analog circuits with AC coupling.

I\+E\+EE 802.\+3 compatible.
