\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_sk_text_blob_builder_1_1_run_buffer}{Run\+Buffer}} supplies storage for glyphs and positions within a run.

A run is a sequence of glyphs sharing font metrics and positioning. Each run may position its glyphs in one of three ways\+: by specifying where the first glyph is drawn, and allowing font metrics to determine the advance to subsequent glyphs; by specifying a baseline, and the position on that baseline for each glyph in run; or by providing \mbox{\hyperlink{struct_sk_point}{Sk\+Point}} array, one per glyph.
