\doxysubsection{Detailed Description}
A base class interface for a bitmap abstraction around the different drawing back end bitmap representations. In most cases it does own the bitmap data, the exception being with Nano\+VG, where the image is loaded onto the G\+PU as a texture, but still needs to be freed. Most of the time end-\/users will deal with \mbox{\hyperlink{class_i_bitmap}{I\+Bitmap}} rather than \mbox{\hyperlink{class_a_p_i_bitmap}{A\+P\+I\+Bitmap}}, which is used behind the scenes.
