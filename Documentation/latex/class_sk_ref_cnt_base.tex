\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_ref_cnt_base}{Sk\+Ref\+Cnt\+Base}} is the base class for objects that may be shared by multiple objects. When an existing owner wants to share a reference, it calls \mbox{\hyperlink{class_sk_ref_cnt_base_abbbd318b36c9e232c70488254e1f5e67}{ref()}}. When an owner wants to release its reference, it calls \mbox{\hyperlink{class_sk_ref_cnt_base_afed65f99f91ef48a7892fafe4829d2f1}{unref()}}. When the shared object\textquotesingle{}s reference count goes to zero as the result of an \mbox{\hyperlink{class_sk_ref_cnt_base_afed65f99f91ef48a7892fafe4829d2f1}{unref()}} call, its (virtual) destructor is called. It is an error for the destructor to be called explicitly (or via the object going out of scope on the stack or calling delete) if get\+Ref\+Cnt() $>$ 1.
