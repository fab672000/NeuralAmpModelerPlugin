Use the system fallback to find a typeface for the given character. Note that bcp47 is a combination of I\+SO 639, 15924, and 3166-\/1 codes, so it is fine to just pass a I\+SO 639 here.

bcp47\mbox{[}0\mbox{]} is the least significant fallback, bcp47\mbox{[}bcp47\+Count-\/1\mbox{]} is the most significant.
