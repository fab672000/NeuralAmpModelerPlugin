\hypertarget{group___result_codes}{}\doxysection{Result Codes}
\label{group___result_codes}\index{Result Codes@{Result Codes}}


Result codes are values returned from methods in an interface. Result codes are bitfield that provide a richer set of information than a simple true or false return value. The result code can be used to determine the exact nature of a failure in the A\+PI.
