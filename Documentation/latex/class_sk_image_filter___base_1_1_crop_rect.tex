Apply this crop\+Rect to the image\+Bounds. If a given edge of the crop\+Rect is not set, then the corresponding edge from image\+Bounds will be used. If \char`\"{}embiggen\char`\"{} is true, the crop rect is allowed to enlarge the size of the rect, otherwise it may only reduce the rect. Filters that can affect transparent black should pass \char`\"{}true\char`\"{}, while all other filters should pass \char`\"{}false\char`\"{}.

Note\+: image\+Bounds is in \char`\"{}device\char`\"{} space, as the output cropped rectangle will be, so the matrix is ignored for those. It is only applied to the crop\+Rect\textquotesingle{}s bounds.
