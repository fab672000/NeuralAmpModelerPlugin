The flanger effect impresses uniformly spaced sweeping notches on the sound spectrum. It accomplishes this using a variable comb filter, which works by mixing the input sound with a variably delayed-\/and-\/scaled copy of the input. Unlike the phaser, the flanger notches are uniformly spaced on a linear frequency scale, and there can be many more of them at low cost.\hypertarget{md__c_1__users_fab_src__github_branches__neural_amp_modeler_plugin_i_plug2__dependencies__build_377ca9432b90e6cd4c1a5390578c5b9f_autotoc_md913}{}\doxysection{Parameters}\label{md__c_1__users_fab_src__github_branches__neural_amp_modeler_plugin_i_plug2__dependencies__build_377ca9432b90e6cd4c1a5390578c5b9f_autotoc_md913}

\begin{DoxyCode}{0}

The result of this compile is a zip file {\ttfamily flanger-\/sam.\+zip} which contains the following three files\+:


\begin{DoxyItemize}
\item {\ttfamily fast\+\_\+pow2.\+h}
\item {\ttfamily sam\+Faust\+D\+S\+P.\+cpp}
\item {\ttfamily sam\+Faust\+D\+S\+P.\+h}
\end{DoxyItemize}

These 3 files should be placed into the directory {\ttfamily sam\+\_\+baremetal\+\_\+framework\+\_\+core1/src/faust}

In addition there is a header file that is common across all cores called {\ttfamily audio\+\_\+system\+\_\+config.\+h}. In this file the following pre-\/processor variables should be set in the following way.
