\doxysubsection{Detailed Description}
Unnamed (process local) counting semaphore.

The civilized person\textquotesingle{}s synchronisation primitive. A counting semaphore is an integer which is always non-\/negative, so, an attempted decrement (or \char`\"{}wait\char`\"{}) will block if the value is 0, until another thread does an increment (or \char`\"{}post\char`\"{}).

At least on Lignux, the main advantage of this is that it is fast and the only safe way to reliably signal from a real-\/time audio thread. The counting semantics also complement ringbuffers of events nicely.
