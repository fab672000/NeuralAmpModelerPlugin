\doxysubsection{Detailed Description}
Default implementation for a non-\/distributable Plug-\/in that combines processor and edit controller in one component.


\begin{DoxyItemize}
\item \mbox{[}released\+: 3.\+0.\+2\mbox{]}
\end{DoxyItemize}

This can be used as base class for a V\+ST 3 effect implementation in case that the standard way of defining two separate components would cause too many implementation difficulties\+:
\begin{DoxyItemize}
\item Cubase 4.\+2 is the first host that supports combined V\+ST 3 Plug-\/ins
\item {\bfseries{ Use this class only after giving the standard way of defining two components serious considerations! }} 
\end{DoxyItemize}
