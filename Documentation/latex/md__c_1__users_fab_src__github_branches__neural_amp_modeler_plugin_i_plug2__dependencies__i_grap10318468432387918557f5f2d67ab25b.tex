\begin{DoxyItemize}
\item Shared buffers between C\+PU and G\+PU.
\item Various Metal states are cached whenever possible.
\item Low overheads compared to Open\+GL.
\item Pre-\/compiled shaders. (no need to compile shaders at runtime)
\item Seamless integration with powerful Metal features such as \href{https://developer.apple.com/documentation/metalperformanceshaders}{\texttt{ Metal Performance Shaders}}.
\end{DoxyItemize}\hypertarget{md__c_1__users_fab_src__github_branches__neural_amp_modeler_plugin_i_plug2__dependencies__i_grap10318468432387918557f5f2d67ab25b_autotoc_md1326}{}\doxysection{Installation}\label{md__c_1__users_fab_src__github_branches__neural_amp_modeler_plugin_i_plug2__dependencies__i_grap10318468432387918557f5f2d67ab25b_autotoc_md1326}
