\doxysubsection{Detailed Description}
Interface for intercepting pre-\/composed layer creation.

Embedders can register interceptors with animation builders, to substitute target layers with arbitrary/externally-\/controlled content (see \mbox{\hyperlink{classskottie_1_1_external_layer}{External\+Layer}} above).
