\doxysubsection{Detailed Description}
A \mbox{\hyperlink{struct_control_ramp}{Control\+Ramp}} describes one value changing over time. It can be easily converted into a signal for more processing, or if sample accuracy is not needed, just the end value can be used. It describes a piecewise function in three pieces\+: from \mbox{[}0, start\+Value\mbox{]} to \mbox{[}transition\+Start, start\+Value\mbox{]} from \mbox{[}transition\+Start, start\+Value\mbox{]} to \mbox{[}transition\+End, end\+Value\mbox{]} from \mbox{[}transition\+End, end\+Value\mbox{]} to \mbox{[}block\+Size, end\+Value\mbox{]}
