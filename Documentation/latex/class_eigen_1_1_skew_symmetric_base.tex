\subsubsection*{template$<$typename Derived$>$\newline
class Eigen\+::\+Skew\+Symmetric\+Base$<$ Derived $>$}

Base class for skew symmetric matrices and expressions. 

This is the base class that is inherited by \mbox{\hyperlink{class_eigen_1_1_skew_symmetric_matrix3}{Skew\+Symmetric\+Matrix3}} and related expression types, which internally use a three vector for storing the entries. Skew\+Symmetric types always represent square three times three matrices.

This implementations follows class \mbox{\hyperlink{class_eigen_1_1_diagonal_matrix}{Diagonal\+Matrix}}
