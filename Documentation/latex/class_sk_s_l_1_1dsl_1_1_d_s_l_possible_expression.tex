\doxysubsection{Detailed Description}
Represents an Expression which may have failed and/or have pending errors to report. Converting a Possible\+Expression into an Expression requires \mbox{\hyperlink{class_sk_s_l_1_1_position_info}{Position\+Info}} so that any pending errors can be reported at the correct position.

Possible\+Expression is used instead of Expression in situations where it is not possible to capture the \mbox{\hyperlink{class_sk_s_l_1_1_position_info}{Position\+Info}} at the time of Expression construction (notably in operator overloads, where we cannot add default parameters).
