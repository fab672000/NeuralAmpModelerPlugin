Unlike C++ classes, {\bfseries{V\+S\+T-\/\+MA}} interfaces do not use inheritance to express specializations of objects. Usually all interfaces are derived from F\+Unknown. This is because interfaces must {\bfseries{never}} change after they have been released. The V\+ST Module Architecture Interfaces use inheritance only for versioning! All specializations will be modeled as separate interfaces!
