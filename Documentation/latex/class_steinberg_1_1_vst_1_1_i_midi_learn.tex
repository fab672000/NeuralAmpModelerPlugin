If this interface is implemented by the edit controller, the host will call this method whenever there is live M\+I\+D\+I-\/\+CC input for the plug-\/in. This way, the plug-\/in can change its M\+I\+D\+I-\/\+CC parameter mapping and inform the host via the \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_component_handler_a65e162e3686dbcd5f2739b6481f69186}{I\+Component\+Handler\+::restart\+Component}} with the k\+Midi\+C\+C\+Assignment\+Changed flag. Use this if you want to implement custom M\+I\+D\+I-\/\+Learn functionality in your plug-\/in.
