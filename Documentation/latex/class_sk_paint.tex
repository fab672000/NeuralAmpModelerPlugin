\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_paint}{Sk\+Paint}} controls options applied when drawing. \mbox{\hyperlink{class_sk_paint}{Sk\+Paint}} collects all options outside of the \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} clip and \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} matrix.

Various options apply to strokes and fills, and images.

\mbox{\hyperlink{class_sk_paint}{Sk\+Paint}} collects effects and filters that describe single-\/pass and multiple-\/pass algorithms that alter the drawing geometry, color, and transparency. For instance, \mbox{\hyperlink{class_sk_paint}{Sk\+Paint}} does not directly implement dashing or blur, but contains the objects that do so.
