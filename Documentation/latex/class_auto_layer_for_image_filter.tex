\doxysubsection{Detailed Description}
We implement Image\+Filters for a given draw by creating a layer, then applying the imagefilter to the pixels of that layer (its backing surface/image), and then we call restore() to xfer that layer to the main canvas.


\begin{DoxyEnumerate}
\item Save\+Layer (with a paint containing the current imagefilter and xfermode)
\item Generate the src pixels\+: Remove the imagefilter and the xfermode from the paint that we (Auto\+Draw\+Looper) return (f\+Paint). We then draw the primitive (using srcover) into a cleared buffer/surface.
\item Restore the layer created in \#1 The imagefilter is passed the buffer/surface from the layer (now filled with the src pixels of the primitive). It returns a new \char`\"{}filtered\char`\"{} buffer, which we draw onto the previous layer using the xfermode from the original paint. 
\end{DoxyEnumerate}
