Main audio engine for the Mega\+Drone sample. It is responsible for\+:


\begin{DoxyItemize}
\item Creating the callback object which will be supplied when constructing the audio stream
\item Setting the C\+PU core I\+Ds to which the callback thread should bind to
\item Creating the playback stream, including setting the callback object
\item Creating {\ttfamily \mbox{\hyperlink{class_synth}{Synth}}} which will render the audio inside the callback
\item Starting the playback stream
\item Restarting the playback stream when {\ttfamily restart()} is called by the callback object
\end{DoxyItemize}
