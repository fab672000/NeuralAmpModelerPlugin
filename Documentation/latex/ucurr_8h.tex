\doxysubsection{Detailed Description}
C A\+PI\+: Encapsulates information about a currency. 

The ucurr A\+PI encapsulates information about a currency, as defined by I\+SO 4217. A currency is represented by a 3-\/character string containing its I\+SO 4217 code. This A\+PI can return various data necessary the proper display of a currency\+:


\begin{DoxyItemize}
\item A display symbol, for a specific locale 
\item The number of fraction digits to display 
\item A rounding increment 
\end{DoxyItemize}

The {\ttfamily Decimal\+Format} class uses these data to display currencies. \begin{DoxyAuthor}{Author}
Alan Liu 
\end{DoxyAuthor}
