\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Scalar\+\_\+$>$\newline
class Eigen\+::\+Diagonal\+Preconditioner$<$ Scalar\+\_\+ $>$}

A preconditioner based on the digonal entries. 

This class allows to approximately solve for A.\+x = b problems assuming A is a diagonal matrix. In other words, this preconditioner neglects all off diagonal entries and, in \mbox{\hyperlink{namespace_eigen}{Eigen}}\textquotesingle{}s language, solves for\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{A.diagonal().asDiagonal() . x = b}
\end{DoxyCode}



\begin{DoxyTemplParams}{Template Parameters}
{\em Scalar\+\_\+} & the type of the scalar.\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

This preconditioner is suitable for both selfadjoint and general problems. The diagonal entries are pre-\/inverted and stored into a dense vector.

\begin{DoxyNote}{Note}
A variant that has yet to be implemented would attempt to preserve the norm of each column.
\end{DoxyNote}
