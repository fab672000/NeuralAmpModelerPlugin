\doxysubsection{Detailed Description}
Edit controller component interface extension\+: \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_parameter_function_name}{Vst\+::\+I\+Parameter\+Function\+Name}}

This interface allows the host to get a parameter associated to a specific meaning (a function\+Name) for a given unit. The host can use this information, for example, for drawing a Gain Reduction meter in its own \mbox{\hyperlink{class_u_i}{UI}}. In order to get the plain value of this parameter, the host should use the \mbox{\hyperlink{class_steinberg_1_1_vst_1_1_i_edit_controller_ab359d3429a9a48d25f94d32ecd22011c}{I\+Edit\+Controller\+::normalized\+Param\+To\+Plain}}. The host can automatically map parameters to dedicated \mbox{\hyperlink{class_u_i}{UI}} controls, such as the wet-\/dry mix knob or Randomize button.\hypertarget{class_steinberg_1_1_vst_1_1_i_parameter_function_name_IParameterFunctionNameExample}{}\doxysubsection{Example}\label{class_steinberg_1_1_vst_1_1_i_parameter_function_name_IParameterFunctionNameExample}
