\doxysubsection{Detailed Description}
Base class for various oscillators. The oscillator has a phase that ranges from -\/1.\+0 to +1.\+0. That makes it easier to implement simple algebraic waveforms.

Subclasses must implement \mbox{\hyperlink{classflowgraph_1_1_flow_graph_node_a343767f9185a9cf23003c7fa8382b7fd}{on\+Process()}}.

This module has \char`\"{}frequency\char`\"{} and \char`\"{}amplitude\char`\"{} ports for control.
