\doxysubsection{Detailed Description}
Container for packet-\/related data. 

This class collects a number of packet-\/related data into the same object and provides a facility for tracking when the parameter is \char`\"{}dirty\char`\"{}, i.\+e. after its value has been updated and before an associated packet has not been posted.
