\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_picture_1_1_abort_callback}{Abort\+Callback}} is an abstract class. An implementation of \mbox{\hyperlink{class_sk_picture_1_1_abort_callback}{Abort\+Callback}} may passed as a parameter to \mbox{\hyperlink{class_sk_picture_a9e5313999fc681ec64586dada8f5a259}{Sk\+Picture\+::playback}}, to stop it before all drawing commands have been processed.

If \mbox{\hyperlink{class_sk_picture_1_1_abort_callback_a07b6acd7495385015c0255818409893b}{Abort\+Callback\+::abort}} returns true, \mbox{\hyperlink{class_sk_picture_a9e5313999fc681ec64586dada8f5a259}{Sk\+Picture\+::playback}} is interrupted.

{\footnotesize\ttfamily virtual bool Sk\+Picture\+::\+Abort\+Callback\+::abort (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [pure virtual]}}

Stops \mbox{\hyperlink{class_sk_picture}{Sk\+Picture}} playback when some condition is met. A subclass of \mbox{\hyperlink{class_sk_picture_1_1_abort_callback}{Abort\+Callback}} provides an override for \mbox{\hyperlink{class_sk_picture_1_1_abort_callback_a07b6acd7495385015c0255818409893b}{abort()}} that can stop \mbox{\hyperlink{class_sk_picture_a9e5313999fc681ec64586dada8f5a259}{Sk\+Picture\+::playback}}.

The part of \mbox{\hyperlink{class_sk_picture}{Sk\+Picture}} drawn when aborted is undefined. \mbox{\hyperlink{class_sk_picture}{Sk\+Picture}} instantiations are free to stop drawing at different points during playback.

If the abort happens inside one or more calls to \mbox{\hyperlink{class_sk_canvas_ac98147dc771b47c7353a51e7c171e428}{Sk\+Canvas\+::save()}}, stack of \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} matrix and \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} clip values is restored to its state before \mbox{\hyperlink{class_sk_picture_a9e5313999fc681ec64586dada8f5a259}{Sk\+Picture\+::playback}} was called.

true to stop playback
