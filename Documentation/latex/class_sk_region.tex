\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sk_region}{Sk\+Region}} describes the set of pixels used to clip \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}}. \mbox{\hyperlink{class_sk_region}{Sk\+Region}} is compact, efficiently storing a single integer rectangle, or a run length encoded array of rectangles. \mbox{\hyperlink{class_sk_region}{Sk\+Region}} may reduce the current \mbox{\hyperlink{class_sk_canvas}{Sk\+Canvas}} clip, or may be drawn as one or more integer rectangles. \mbox{\hyperlink{class_sk_region}{Sk\+Region}} iterator returns the scan lines or rectangles contained by it, optionally intersecting a bounding rectangle.
