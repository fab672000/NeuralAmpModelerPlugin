It performs the following incomplete factorization\+: $ S P A P\textnormal{\textquotesingle} S \approx L L\textnormal{\textquotesingle} $ where L is a lower triangular factor, S is a diagonal scaling matrix, and P is a fill-\/in reducing permutation as computed by the ordering method.

{\bfseries{Shifting}} {\bfseries{strategy\+:}} Let $ B = S P A P\textnormal{\textquotesingle} S $ be the scaled matrix on which the factorization is carried out, and $ \beta $ be the minimum value of the diagonal. If $ \beta > 0 $ then, the factorization is directly performed on the matrix B. Otherwise, the factorization is performed on the shifted matrix $ B + (\sigma+|\beta| I $ where $ \sigma $ is the initial shift value as returned and set by \mbox{\hyperlink{class_eigen_1_1_incomplete_cholesky_afe6b11310e22c1853577746a37fbcdc7}{set\+Initial\+Shift()}} method. The default value is $ \sigma = 10^{-3} $. If the factorization fails, then the shift in doubled until it succeed or a maximum of ten attempts. If it still fails, as returned by the \mbox{\hyperlink{class_eigen_1_1_incomplete_cholesky_a11a53c3787990c0287f7296f72779509}{info()}} method, then you can either increase the initial shift, or better use another preconditioning technique.
