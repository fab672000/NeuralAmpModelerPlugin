intervals\+: array containing an even number of entries ($>$=2), with the even indices specifying the length of \char`\"{}on\char`\"{} intervals, and the odd indices specifying the length of \char`\"{}off\char`\"{} intervals. This array will be copied in Make, and can be disposed of freely after. count\+: number of elements in the intervals array phase\+: offset into the intervals array (mod the sum of all of the intervals).

For example\+: if intervals\mbox{[}\mbox{]} = \{10, 20\}, count = 2, and phase = 25, this will set up a dashed path like so\+: 5 pixels off 10 pixels on 20 pixels off 10 pixels on 20 pixels off ... A phase of -\/5, 25, 55, 85, etc. would all result in the same path, because the sum of all the intervals is 30.

Note\+: only affects stroked paths.
