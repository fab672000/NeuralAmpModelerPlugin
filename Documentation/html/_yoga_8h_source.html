256 // Yoga specific properties, not compatible with flexbox specification Aspect
257 // ratio control the size of the undefined dimension of a node. Aspect ratio is
258 // encoded as a floating point value width/height. e.g. A value of 2 leads to a
259 // node with a width twice the size of its height while a value of 0.5 gives the
260 // opposite effect.
261 //
262 // - On a node with a set width/height aspect ratio control the size of the
263 // unset dimension
264 // - On a node with a set flex basis aspect ratio controls the size of the node
265 // in the cross axis if unset
266 // - On a node with a measure function aspect ratio works as though the measure
267 // function measures the flex basis
268 // - On a node with flex grow/shrink aspect ratio controls the size of the node
269 // in the cross axis if unset
270 // - Aspect ratio takes min/max dimensions into account
271 WIN_EXPORT void YGNodeStyleSetAspectRatio(YGNodeRef node, float aspectRatio);
272 WIN_EXPORT float YGNodeStyleGetAspectRatio(YGNodeConstRef node);
