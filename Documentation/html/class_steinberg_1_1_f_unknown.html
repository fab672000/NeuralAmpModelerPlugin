Detailed Description

The basic interface of all interfaces.

  • The FUnknown::queryInterface method is used to retrieve pointers to other interfaces of the object.
  • FUnknown::addRef and FUnknown::release manage the lifetime of the object. If no more references exist, the object is destroyed in memory.

Interfaces are identified by 16 byte Globally Unique Identifiers. The SDK provides a class called FUID for this purpose.
