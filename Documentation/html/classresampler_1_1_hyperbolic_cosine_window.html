Detailed Description

Calculate a HyperbolicCosineWindow window centered at 0. This can be used in place of a Kaiser window.

The code is based on an anonymous contribution by "a concerned citizen": https://dsp.stackexchange.com/questions/37714/kaiser-window-approximation
