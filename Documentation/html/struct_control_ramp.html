Detailed Description

A ControlRamp describes one value changing over time. It can be easily converted into a signal for more processing, or if sample accuracy is not needed, just the end value can be used. It describes a piecewise function in three pieces: from [0, startValue] to [transitionStart, startValue] from [transitionStart, startValue] to [transitionEnd, endValue] from [transitionEnd, endValue] to [blockSize, endValue]
