void SkImageFilter_Base::CropRect::applyTo ( const SkIRect &  imageBounds,
const SkMatrix &  matrix,
bool  embiggen,

Apply this cropRect to the imageBounds. If a given edge of the cropRect is not set, then the corresponding edge from imageBounds will be used. If "embiggen" is true, the crop rect is allowed to enlarge the size of the rect, otherwise it may only reduce the rect. Filters that can affect transparent black should pass "true", while all other filters should pass "false".

Note: imageBounds is in "device" space, as the output cropped rectangle will be, so the matrix is ignored for those. It is only applied to the cropRect's bounds.
