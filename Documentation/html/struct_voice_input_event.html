Detailed Description

A VoiceInputEvent describes a change in input to be applied to one more more voices. mAddress specifies which voices should receive the change. mAction is the type of property change. mControllerNumber is the controller number to change if mAction is kController. mValue is the new value associated with the change. mSampleOffset is the number of samples into a processing buffer at which the change should occur.
