Detailed Description

A container object for plug-in parameters.

This implementation uses a STL vector to store a plug-in's set of parameters. This class contains a real implementation of the AAXLibraryFeatures_ParameterManager (as opposed to a proxy.)

For more information, see AAXLibraryFeatures_ParameterManager.
