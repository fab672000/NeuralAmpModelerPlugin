Detailed Description

Represents an Expression which may have failed and/or have pending errors to report. Converting a PossibleExpression into an Expression requires PositionInfo so that any pending errors can be reported at the correct position.

PossibleExpression is used instead of Expression in situations where it is not possible to capture the PositionInfo at the time of Expression construction (notably in operator overloads, where we cannot add default parameters).
