Defines a type of acfUID structure.

This macro is used to both declare and define acfUID constants. If the symbol INITACFIDS is defined then the constant is fully defined, otherwise the constant is just declared.

Note
To avoid duplicate symbol definitions the symbol INITACFIDS should only be defined in one source file of an executable module.
