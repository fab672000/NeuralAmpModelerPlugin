Detailed Description

template<class T>
struct Steinberg::FNuller< T >

Template definition for classes that help resetting an object's value. A stack allocated object of this type automatically resets the value of an at construction time passed object to null when it reaches the end of its scope.

Intended usage:

int theObject
= 0;
{
Steinberg::FNuller<int> theNuller (theObject);
theObject = 1;
} // Here the destructor of theNuller resets the value of theObject to 0.
