Detailed Description

A parametrized line.

\geometry_module

A parametrized line is defined by an origin point and a unit direction vector such that the line corresponds to the set , .

Template Parameters
Scalar_the scalar type, i.e., the type of the coefficients
AmbientDim_the dimension of the ambient space, can be a compile time value or Dynamic.
