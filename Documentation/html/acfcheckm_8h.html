◆ THROWRESULT

#define THROWRESULT   throw

Defining THROWACFRESULT, allows or supress acfcheckmM to throwing a failed ACFRESULT. The defualt prevents throwing in a debug build. Because a message box is enough warning when debugging... A defualt release build will throw any result that represents failure.

Enabling provides for behaviour similar to acfcheck which does throw a result. In a release build the default behaviour is to throw
