void flowgraph::FlowGraphNode::setDataPulledAutomatically ( bool  automatic)

Set true if you want the data pulled through the graph automatically. This is the default.

Set false if you want to pull the data from the input ports in the onProcess() method. You might do this, for example, in a sample rate converting node.
