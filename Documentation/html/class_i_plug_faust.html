Detailed Description

This abstract interface is used by the IPlug FAUST architecture file and the IPlug libfaust JIT compiling class FaustGen In order to provide a consistent interface to FAUST DSP whether using the JIT compiler or a compiled C++ class

Member Function Documentation

◆ CompileCPP()

static bool IPlugFaust::CompileCPP ( )
inlinestatic

Call this method from FaustGen in order to execute a shell command and compile the C++ code against the IPlugFaust_arch architecture file There is a NO-OP implementation here so that when not using the JIT compiler, the same class can be used interchangeably

Returns
true on success

◆ GetDrawPath()

virtual void IPlugFaust::GetDrawPath ( WDL_String &  path)
inlinevirtual

In FaustGen this is implemented, so that the SVG files generated by a specific instance can be located. The path to the SVG file for process.svg will be returned. There is a NO-OP implementation here so that when not using the JIT compiler, the same class can be used interchangeably

Parameters
pathThe absolute path to process.svg for this instance.
