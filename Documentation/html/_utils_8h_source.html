13 // This struct is an helper model to hold the data for step 4 of flexbox algo,
14 // which is collecting the flex items in a line.
15 //
16 // - itemsOnLine: Number of items which can fit in a line considering the
17 // available Inner dimension, the flex items computed flexbasis and their
18 // margin. It may be different than the difference between start and end
19 // indicates because we skip over absolute-positioned items.
20 //
21 // - sizeConsumedOnCurrentLine: It is accumulation of the dimensions and margin
22 // of all the children on the current line. This will be used in order to
23 // either set the dimensions of the node if none already exist or to compute
24 // the remaining space left for the flexible children.
25 //
26 // - totalFlexGrowFactors: total flex grow factors of flex items which are to be
27 // layed in the current line
28 //
29 // - totalFlexShrinkFactors: total flex shrink factors of flex items which are
30 // to be layed in the current line
31 //
32 // - endOfLineIndex: Its the end index of the last flex item which was examined
33 // and it may or may not be part of the current line(as it may be absolutely
34 // positioned or including it may have caused to overshoot availableInnerDim)
35 //
36 // - relativeChildren: Maintain a vector of the child nodes that can shrink
37 // and/or grow.
38 
39 struct YGCollectFlexItemsRowValues {
40  uint32_t itemsOnLine;
41  float sizeConsumedOnCurrentLine;
42  float totalFlexGrowFactors;
43  float totalFlexShrinkScaledFactors;
44  uint32_t endOfLineIndex;
45  std::vector<YGNodeRef> relativeChildren;
46  float remainingFreeSpace;
47  // The size of the mainDim for the row after considering size, padding, margin
48  // and border of flex items. This is used to calculate maxLineDim after going
49  // through all the rows to decide on the main axis size of owner.
50  float mainDim;
51  // The size of the crossDim for the row after considering size, padding,
52  // margin and border of flex items. Used for calculating containers crossSize.
53  float crossDim;
54 };
