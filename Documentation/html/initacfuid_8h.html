Detailed Description

Include initacfuid.h to enable acfUID initialization. This must be done once per exe/dll. After this file, include one or more of the acfUID definition files.

Defines a uid structure of type acfUID and initializes member data. Generally you will want to use this macro to define all uid's within a definition.

Warning
The uid must be unique.
Example: 
DEFINE_ACFUID(acfUID, ACF_MyEffectUID, 0x51b30d8a, 0xf54, 0x4092, 0xba, 0x41, 0x46, 0xdc, 0x91, 0xee, 0xf7, 0xf4);
