int  expectedInputs 
)

Attempt to unflatten the cropRect and the expected number of input filters. If any number of input filters is valid, pass -1. If this fails (i.e. corrupt buffer or contents) then return false and common will be left uninitialized. If this returns true, then inputCount() is the number of found input filters, each of which may be NULL or a valid imagefilter.
