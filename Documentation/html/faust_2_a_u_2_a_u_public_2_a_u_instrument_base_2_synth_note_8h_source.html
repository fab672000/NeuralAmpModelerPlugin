70 /*
71  This table describes the state transitions for SynthNotes
72 
73  EVENT CURRENT STATE NEW STATE
74  note on free attacked
75  note off attacked (and sustain on) released but sustained
76  note off attacked released
77  note off sostenutoed released but sostenutoed
78  sustain on -- no changes --
79  sustain off released but sustained released
80  sostenuto on attacked sostenutoed
81  sostenuto off sostenutoed attacked
82  sostenuto off released but sostenutoed (and sustain on) released but sustained
83  sostenuto off released but sostenutoed released
84  end of note any state free
85  soft voice stealing any state fast released
86  hard voice stealing any state free
87 
88  soft voice stealing happens when there is a note on event and NumActiveNotes > MaxActiveNotes
89  hard voice stealing happens when there is a note on event and NumActiveNotes == NumNotes (no free notes)
90  voice stealing removes the quietest note in the highest numbered state that has sounding notes.
91 */
