25 /*
26  * This class is intended to be used as:
27  * Get an SkSurfaceCharacterization representing the intended gpu-backed destination SkSurface
28  * Create one of these (an SkDeferredDisplayListRecorder) on the stack
29  * Get the canvas and render into it
30  * Snap off and hold on to an SkDeferredDisplayList
31  * Once your app actually needs the pixels, call SkSurface::draw(SkDeferredDisplayList*)
32  *
33  * This class never accesses the GPU but performs all the cpu work it can. It
34  * is thread-safe (i.e., one can break a scene into tiles and perform their cpu-side
