T  timeScalar = 1. 
)

Retrigger the envelope. This method will cause the envelope to move to a "releasedToRetrigger" stage, which is a fast ramp to zero in RETRIGGER_RELEASE_TIME, used when voices are stolen to avoid clicks.

Parameters
newStartLevelthe overall level when the envelope restarts (usually linked to MIDI velocity)
timeScalarFactor to scale the envelope's rates. Use this, for example to adjust the envelope stage rates based on the key pressed
