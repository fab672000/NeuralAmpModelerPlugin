Detailed Description

Represents a Statement which may have failed and/or have pending errors to report. Converting a PossibleStatement into a Statement requires PositionInfo so that any pending errors can be reported at the correct position.

PossibleStatement is used instead of Statement in situations where it is not possible to capture the PositionInfo at the time of Statement construction.
