acfcheck is a convenience function used for validating the success of returning ACFRESULT codes.

acfcheck( result )

Remarks
if the return code is a comprised of a ACFFAILED code this function will throw an ACFRESULT exception. For this reason, use of the function should be wrapped in a try – catch block that can handle the exception, such as the BEGIN_ACF_METHOD and END_ACF_METHOD macros.
