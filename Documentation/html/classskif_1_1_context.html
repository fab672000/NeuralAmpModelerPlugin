bool skif::Context::isValid ( ) const
inline

Since a context can be built directly, its constructor has no chance to "return null" if it's given invalid or unsupported inputs. Call this to know of the the context can be used.

The SkImageFilterCache Key, for example, requires a finite ctm (no infinities or NaN), so that test is part of isValid.
